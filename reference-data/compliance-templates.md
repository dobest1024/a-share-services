# 合规话术模板

所有 A 股 agent 的输出必须使用本文档的标准话术。**禁止**自创"投资建议"措辞。

## 1. 标准免责声明（每份输出必带）

> ⚠️ 本输出仅为分析框架与参考观点，**不构成任何投资建议**。数据来源已标注；结论需由具备相应资质的专业人士复核。使用者需自行遵守中国证监会、交易所及相关法律法规。市场有风险，投资需谨慎。

## 2. 措辞替换表

| ❌ 禁用 | ✅ 改用 |
|---|---|
| 建议买入 / 卖出 / 增持 / 减持 | 关注要点 / 观察方向 |
| 目标价 XX 元 | 估值区间参考 / 历史估值分位 |
| 潜在涨幅 XX% | 估值修复空间（在 ... 假设下） |
| 一定会 / 必然 / 肯定 | 倾向于 / 概率较大 / 历史上 |
| 强烈推荐 | 重点跟踪 |
| 必涨 / 必跌 | 上行/下行风险因素 |
| 内幕消息 / 一手信息 | 公开披露信息（附公告链接） |

## 3. 个人投资者 / 投顾客户输出额外规则

针对 `retail-advisory` 与 `advisor-meeting-prep` agent：

1. **不指定个股权重** — 只输出资产配置框架（股 / 债 / 商品 / 现金的比例区间）。个股层面只做"标的池"，不做"权重表"。
2. **必须人工签字** — 任何对外发送的内容必须由具备投顾资质的人员复核签字。agent 输出末尾固定提示该流程。
3. **客户类型确认** — 输出前要求确认客户已完成投资者适当性匹配。

## 4. 数据引用规范

- 价格 / 行情数据：标注 `[akshare YYYY-MM-DD]` 或 `[tushare YYYY-MM-DD]`
- 公告内容：附巨潮链接 `http://www.cninfo.com.cn/...`
- 政策原文：附政府/部委官网链接，禁止间接引用
- 无法验证的市场传闻：明确标 `[未核实]` 或直接不写入输出

## 5. 输出末尾固定段落

```
---
> ⚠️ **本输出未经合规审核与人工复核，不得直接对外发布。**
> 数据来源与方法：{此处列出本次使用的 MCP / skill}
> 生成时间：{YYYY-MM-DD HH:mm}
```
