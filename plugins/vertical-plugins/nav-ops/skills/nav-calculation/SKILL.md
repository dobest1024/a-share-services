# 基金净值核算

## 核算流程

### 公募基金
每日 15:00 收盘后，由托管行做净值核算（基金管理人独立复核）：

1. 估值（收盘后）
   - 股票：当日收盘价
   - 债券：估值机构提供（中债估值 / 中证估值）
   - 货币：摊余成本法（部分公募）/ 市值法
   - 衍生品：盯市估值
2. 计提
   - 管理费（按日计提，年化 0.5-1.5%）
   - 托管费（按日计提，年化 0.1-0.3%）
   - 销售服务费（部分产品）
3. 计算
   - 当日总资产
   - 减负债（待付申购款 / 待付赎回款 / 应付费用）
   - 总资产净值 = 总资产 - 总负债
   - 单位净值 = 总资产净值 / 基金份额

### 私募基金
- 一般按月核算（月末最后一个交易日）
- 也有周度净值
- 估值方法相对灵活，但要在合同中明确

## 关键勾稽

### 与上日勾稽
```
今日净值 = 昨日净值 × (1 + 当日组合收益率) - 当日费率
```

### 与持仓勾稽
- 持仓股票市值 = Σ(持仓股数 × 收盘价)
- 持仓债券市值 = 第三方估值
- 持仓现金 = 银行账户余额
- 应收应付 = 待结算款项

### 与申赎勾稽
- 申购份额 = 申购金额 / 当日净值（剔除手续费）
- 赎回份额 × 当日净值 = 赎回金额（剔除手续费）

## 常见差错

### 估值差错
- 停牌股票的估值方法（按行业指数法 / 长期停牌按合理价值）
- 公允价值变动损益
- 持仓股权分置 / 流通 / 限售的差异

### 申赎确认时点
- T 日 15:00 前申购 → T 日净值；之后 → T+1 日净值
- 跨节假日处理

### 费用计提
- 业绩报酬高水位法
- 业绩提取触发时点
- 浮动费率

## A 股相关特殊处理

### 涨跌停股票
- 涨停封单仍按收盘价（涨停价）估值
- 跌停同理
- 异常停牌按净资产 / 行业指数估值

### ST 股
- 涨跌幅限制 5%
- 估值正常按收盘价

### 北交所
- 单独估值规则（涨跌幅 30%）
- 流动性影响估值的合理性

### 港股通
- 港股通持仓按当日港股收盘价
- 汇率使用人民币 / 港币中间价

## 输出结构（净值核算复核报告）

```
## 基金净值核算复核 - {基金代码} - {YYYY-MM-DD}

### 净值快照
- 单位净值：X.XXXX 元
- 累计净值：X.XXXX 元
- 当日涨跌幅：+/-X.XX%
- 当日组合收益贡献：+/-X.XX%

### 持仓估值
- 股票：X 万元（占 X%）
- 债券：X 万元（占 X%）
- 现金：X 万元（占 X%）
- 其他：X 万元

### 主要持仓异常
- {标的}：停牌 / 涨跌停 / 业绩雷 → 估值方式
- ...

### 当日申赎
- 申购：X 份（X 万元）
- 赎回：X 份（X 万元）

### 费率
- 管理费当日计提：X 元
- 托管费当日计提：X 元
- 业绩报酬（如适用）：X 元

### 勾稽校验
- 与上日勾稽：✅ / ❌（差异原因）
- 与持仓勾稽：✅ / ❌
- 与申赎勾稽：✅ / ❌

### 异常提示
- 持仓集中度变化
- 大额申赎冲击
- 流动性紧张标的

### ⚠️ 复核要求
- 本输出为复核辅助
- 净值发布须由合规与运营部门最终确认
```
