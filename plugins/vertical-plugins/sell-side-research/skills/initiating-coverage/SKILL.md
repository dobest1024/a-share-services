# 首推报告（深度首次覆盖）

## 标准结构（30-60 页）

### 1. 投资要点（首页 1 页）
- 一句话定位：公司是什么 + 为什么现在看
- 3-5 个核心论点
- 估值参考区间（**不出目标价、不出评级**，仅参考观点）
- 主要风险

### 2. 公司概况
- 历史沿革（成立 / 上市 / 重大转型）
- 股权结构（实控人 / 一致行动人 / 重要股东）
- 主营业务结构（按业务线 / 产品 / 地区）
- 核心管理层（董事长 / 总经理 / 财务总监 / 关键研发负责人）

### 3. 行业分析
- 行业空间（TAM / 增速 / 渗透率）
- 行业结构（CR3 / CR5 集中度 / 竞争格局）
- 产业链位置（上游议价 / 下游议价）
- 政策环境
- 行业关键变量与拐点

### 4. 公司竞争优势
**护城河 5 维分析**：
- 品牌 / 渠道 / 技术 / 成本 / 网络效应
- 与主要竞争对手的差异化

### 5. 财务分析
- 历史 5 年三表回顾
- 关键指标分解（ROE 杜邦拆解）
- 财务质量扫描
- 现金流与盈利能力匹配度

### 6. 财务预测（未来 3 年）
- 关键假设
- 三表预测（触发 `3-statement-model-cas`）
- 敏感性分析

### 7. 估值
- 主估值方法选择理由
- 估值结果 + 区间
- 历史估值分位
- 同业横截面对比

### 8. 风险因素
- 业务风险（具体到产品 / 客户 / 供应商）
- 财务风险
- 政策 / 行业风险
- 公司治理风险
- 流动性风险

### 9. 附录
- 财报附表
- 业务数据时间序列
- 主要假设说明
- 分析师声明 + 合规提示

## A 股首推合规要求

⚠️ **本仓库 agent 输出仅为内部底稿，不可直接对外发布**。正式卖方研报需满足：

- 分析师持证（CFA / 证券分析师执业证书）
- 评级与目标价的内控审批
- 合规审核签字
- 利益冲突披露（持仓 / 关联方 / 投行业务关系）
- 数据 / 图表来源标注

本 skill 用于辅助生成"研报底稿"，正式发布需经合规与质控团队人工审核。

## 工作流

1. 触发 stock-researcher agent 拿到个股研究框架（基础底稿）
2. 触发 sector-researcher 拿到行业分析
3. 在底稿上展开：竞争优势深度、财务预测、估值
4. 输出未经合规审核的草稿，明确标注"待合规审核"
