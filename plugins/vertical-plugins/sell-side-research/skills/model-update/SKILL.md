# 模型更新

## 触发场景

每次正式财报披露后（年报 / 中报 / 季报）需要更新模型：
1. 用实际数披露替换前期预测
2. 评估假设是否需要修正
3. 调整未来期预测
4. 重算估值

## 标准流程

### Step 1 数据替换
- 把模型中 {当前报告期} 的预测列改为实际值
- 单元格颜色：预测（蓝色）→ 实际（黑色）
- 检查勾稽：三表勾稽是否仍 OK

### Step 2 假设偏离分析
对每个关键假设列出"预测值 / 实际值 / 偏离 %"：
- 收入：分业务对照
- 毛利率
- 销售费用率 / 管理费用率 / 研发费用率
- 资本开支
- 营运资金
- 有效税率

任何偏离 > 5% 的假设需要：
- 找出偏离原因
- 判断是一次性还是结构性
- 决定未来期是否调整

### Step 3 未来期预测调整
- 一次性事件 → 不调整未来
- 结构性变化 → 调整未来期假设
- 行业景气变化 → 全行业模型联动调整

### Step 4 估值更新
- 重新跑 DCF / PB-ROE / 可比估值
- 记录估值区间变化（不出目标价）

### Step 5 留痕
- 模型版本号 +1
- 更新日志：日期 / 更新人 / 主要变化
- 历史快照保留（Excel "另存为带日期"）

## A 股特殊处理

- 一季报 / 三季报披露的是简版数据，没有附注。预测调整慎重
- 半年报 / 年报披露后可大幅调整
- 业绩快报 → 等正式财报后再调（快报数据可能修正）
- 业绩预告 → 仅做"参考性调整"，不直接替换

## 输出结构

```
## {标的} 模型更新 - {报告期} 财报后

### 关键假设偏离
| 假设 | 预测 | 实际 | 偏离 | 原因 | 是否调整未来 |

### 未来 3 年预测调整
| 年份 | 旧收入预测 | 新收入预测 | 旧归母预测 | 新归母预测 |

### 估值变化
- DCF 每股价值：X.XX → Y.YY 元（不构成目标价）
- PE 估值参考区间：[L1, H1] → [L2, H2]

### 关键变化原因
- 1. 毛利率超预期（X% vs 预测 Y%），主要因 ...
- 2. ...

### 后续观察
- 待跟踪指标
- 下次模型更新时点（下次披露后）
```
