# 晨会纪要

## 卖方版（研究所晨会）

### 标准结构（< 2000 字）

```
# {YYYY-MM-DD} 晨会纪要

## 隔夜外盘
- 美股三大指数 / 欧股 / 大宗 / 美债 / 美元指数
- 关键事件（美联储讲话 / 关键数据 / 地缘）

## A 股前日复盘
- 三大指数（沪深主板 / 创业板 / 科创板）涨跌
- 成交量与北向
- 涨跌停潮、连板梯队
- 主线行业 / 风格

## 今日关键事件
- 经济数据（CPI / PMI / 社融 / 出口）
- 政策动态（央行 / 证监会 / 行业部委）
- 重要会议
- 重要财报披露
- 限售解禁

## 重点行业 / 公司推荐
- 行业 1：观点（基于哪份报告）
- 公司 1：跟踪要点（不出操作建议）
- ...

## 今日关注
- 3-5 条最重要的事件 / 数据
```

## 买方版（基金经理晨会）

### 与卖方版的差异
- 加上**自家持仓相关跟踪**
- 加上**操作信号**（仅供基金经理参考，不对外）
- 减少全市场综述，加大重点池子跟踪深度

### 标准结构

```
# {YYYY-MM-DD} 投研晨会

## 隔夜与今日宏观
（同卖方版）

## 持仓个股动态
- 重仓股 1：昨日表现 / 今日预期事件 / 风险提示
- 重仓股 2：...
- 关注股 1：...

## 关注池子
- 行业 1：本周关注哪些个股 / 触发条件
- 行业 2：...

## 操作建议（内部）
- 加仓 / 减仓 / 观望候选（仅基金经理审阅）
- 触发条件 / 价位区间

## 风险监控
- 持仓集中度
- 行业暴露
- 因子暴露（动量 / 价值 / 规模 / 流动性）

## 调研安排
- 今日 / 本周公司调研 / 专家访谈
```

## 工作流

1. 拉取宏观数据（`akshare-mcp.macro.*`）
2. 拉取前日行情综述（`akshare-mcp.stock.market_summary`）
3. 拉取今日财经日历（`akshare-mcp.event.calendar`）
4. 触发 `event-hunter` skill 拿到事件汇总
5. （买方版）拉取持仓清单，跟踪持仓变动
6. 按结构生成晨会纪要

## 合规

- 卖方版若要对外发布，需走合规流程
- 买方版仅供内部使用
- 任何"操作建议"在面向投资者输出时必须删除或改为"参考观点"
