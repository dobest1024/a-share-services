# 行业综述

## 框架

### 1. 行业定义与边界
- 申万分类（一级 / 二级 / 三级）
- 业务实质边界（如"汽车整车 vs 汽车零部件 vs 智能驾驶"）
- 不要混淆"行业"与"主题"（中特估、专精特新是主题不是行业）

### 2. 行业空间
- TAM（总市场容量）/ SAM（可服务市场）/ SOM（可获取市场）
- 历史 5-10 年 CAGR + 未来 3 年预期
- 数据源：行业协会 / 统计局 / 第三方数据公司

### 3. 产业链
- 上游：原材料 / 设备 / 关键技术（议价权）
- 中游：制造 / 加工
- 下游：终端客户（C 端 / B 端 / G 端）
- 价值分布：哪个环节最赚钱

### 4. 竞争格局
- CR3 / CR5 集中度
- 行业前 10 名（市场份额 / 增速 / 利润率）
- 进入壁垒（技术 / 资金 / 牌照 / 渠道）
- 替代品威胁

### 5. 行业经济性
- 商业模式（一次性 / 订阅 / 平台 / 项目制）
- 现金流特征
- 周期性（需求周期 / 产能周期 / 库存周期）
- ROIC 中枢

### 6. 政策环境
- 关键监管部委（工信部 / 发改委 / 央行 / 证监会 / 行业部委）
- 主要政策文件回顾（近 3 年）
- 国家战略对接（如新质生产力对接科技制造）
- 补贴 / 退坡 / 集采等专项政策

### 7. 行业指数与估值
- 申万一级 / 中信一级行业指数
- 行业 PE-TTM / PB / ROE 时间序列
- 历史估值分位
- 与沪深 300 的相对估值

### 8. 重点公司清单
- 龙头股（CR3 / CR5）
- 细分领域龙头
- 新兴公司（成长性）
- 估值矩阵（市值 / 收入 / 利润 / 估值）

### 9. 投资逻辑
- **不出"看多 / 看空"评级**
- 列出"看多论据" + "看空论据" + "关键变量"
- 跟踪指标清单

## A 股行业研究特殊点

### 政策驱动强
- A 股许多行业由政策主导（光伏、新能源车、半导体、医药集采）
- 行业分析必须有政策维度
- 政策预期与落地节奏比基本面更影响短期股价

### 主题与行业交叉
- 一个公司可能属于"汽车"行业（申万）+ "智能驾驶"主题（概念板块）+ "国产替代"政策方向
- 三个口径不冲突，分析时分层处理

### 周期与成长划分
- A 股估值体系对"周期股" vs "成长股"差异巨大
- 同一行业的不同子行业可能一周期一成长（如电子里的半导体设计 vs 面板）

## 工作流

1. 锁定申万二级行业
2. 拉行业指数与估值（`tushare-mcp.sw_industry`）
3. 拉行业政策文件清单（`policy-themes` vertical）
4. 拉行业内 Top 公司财务（`akshare-mcp.fundamental.industry_comps`）
5. 输出 8 段式综述
