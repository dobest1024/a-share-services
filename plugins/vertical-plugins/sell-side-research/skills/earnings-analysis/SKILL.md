# 季报 / 中报 / 年报分析

## 报告期类型与时点

| 报告 | 披露窗口 | 数据完整度 |
|---|---|---|
| 一季报 | 4 月底 | 利润表 + 资产负债表（简版） |
| 中报 | 8 月底 | 完整三表 + 审计鉴证 |
| 三季报 | 10 月底 | 利润表 + 资产负债表（简版） |
| 年报 | 次年 4 月底 | 完整三表 + 审计 + 附注 + 治理 |

## 分析框架

### 1. 收入端
- 同比 / 环比
- 与卖方一致预期对比
- 量价拆分（消费 / 能源）/ 业务线拆分（B2B）/ 地区拆分（出海）
- 应收增速 vs 收入增速（收入质量）

### 2. 利润端
- 毛利率：环比 + 同比 + 与历史中枢对比
- 三项费用率：销售 / 管理 / 研发（注意研发资本化）
- 营业利润率 vs 净利润率 → 非经常性损益影响
- 扣非归母（去掉补助、处置、公允价值）才是看主业

### 3. 资产负债表
- 资产负债率
- 货币资金 vs 有息负债
- 应收 / 存货 / 应付周转
- 商誉 / 净资产
- 关联往来款

### 4. 现金流量表
- 经营性现金流 / 净利润（应 > 0.8 为健康）
- 经营性现金流 / 营业利润
- 自由现金流 = 经营 - 投资性资本开支
- 销售商品收到的现金 / 营业收入（含税计算，应 > 1.0）

### 5. 经营数据（行业特有）
- 消费：单店收入 / 渠道结构 / 库存周转
- 制造：产能利用率 / 在建工程进度
- TMT：DAU/MAU / ARPU / 留存
- 金融：净息差 / 不良率 / 拨备覆盖率
- 医药：研发管线进度 / 集采影响

### 6. 后续展望
- 公司在管理层讨论与分析（MD&A）中的表述
- 下季度业务展望
- 重大经营事件（新工厂 / 新产品 / 新合作）

## 工作流

1. 拉公司财报（`akshare-mcp.fundamental.income / balance / cashflow`）
2. 拉一致预期（`tushare-mcp.consensus.estimate`）
3. 拉财报 PDF（`cninfo-mcp.filing_text`）抽取 MD&A
4. 同业横截面对比
5. 触发 `financial-quality-check` skill
6. 输出"超预期分析 + 趋势判断 + 风险点 + 跟踪问题"
