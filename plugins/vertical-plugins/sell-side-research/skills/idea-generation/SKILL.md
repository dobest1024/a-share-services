# 选股 / 想法生成

## 三种筛选范式

### 1. 多因子筛选（量化）
按因子条件交集出股票池：

**估值因子**
- PE-TTM < X 且 > 0
- PB < X
- 股息率 > X%

**质量因子**
- ROE-TTM > X%（连续 3 年）
- 经营性现金流 / 净利润 > 0.8
- 资产负债率 < X%
- 商誉 / 净资产 < 10%

**增长因子**
- 收入近 3 年 CAGR > X%
- 归母净利润最近 4 季度增速 > X%
- 未出现单季度同比负增长

**规模 / 流动性**
- 自由流通市值 > X 亿
- 日均成交 > X 万元

**风格 / 主题**
- 申万一级 / 二级行业
- 概念板块归属

### 2. 主题筛选
基于政策主题映射 `reference-data/policy-keywords.yaml`：
1. 用户输入主题（如"新质生产力"）
2. 查映射表得到受益申万一级行业
3. 在这些行业内套用多因子筛选

### 3. 事件驱动筛选
基于近期事件触发：
- 业绩超预期 + 卖方一致预期上调
- 北向资金连续加仓
- 龙虎榜机构席位密集
- 大股东 / 高管增持
- 回购公告 + 用途为注销
- 重要客户 / 订单签约

## A 股选股特别考虑

### 风格切换风险
- 不要在因子全为"价值"或全为"成长"时孤注一掷
- 2021 后 A 股小盘风格强，大小盘超额轮动需关注

### 主题轮动节奏
- 主题股波动大、持续期短
- 强调"标的池" 而非"组合"，组合需要再做权重决策

### 流动性约束
- 自由流通市值 < 30 亿的标的对机构资金量不友好
- 个人投资者可适当下沉

## 工作流

1. 用户明确筛选意图（量化 / 主题 / 事件）
2. 设定参数（行业 / 阈值 / 时间窗口）
3. 拉数据（`tushare-mcp` 因子数据为主）
4. 出标的池
5. 对每只展开简易扫描（业务 / 估值 / 财务质量）
6. 输出"标的池表格 + 风险提示"

## 输出结构

```
## 选股结果 - {筛选意图}

### 筛选条件
- 范围：{申万行业 / 全市场}
- 因子：{条件清单}
- 时间窗口：{...}

### 命中数：N

### 标的池
| 代码 | 公司 | 行业 | 市值 | 收入增速 | 归母增速 | ROE | PE | 命中维度 |

### 简易扫描（Top 10）
对每只标注：
- 业务一句话
- 当前估值水位
- 财务质量是否触警
- 近期催化剂

### ⚠️ 风险提示
- 本筛选基于历史数据，未对未来作判断
- 不构成投资建议
- 重大投资决策需人工尽调
```
