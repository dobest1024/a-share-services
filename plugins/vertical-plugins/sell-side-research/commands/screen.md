# /screen {筛选意图}

按用户意图选股，输出标的池。

## 三种范式
- **多因子**：`/screen 低估值 + 高 ROE + 高分红`
- **主题**：`/screen 新质生产力`
- **事件**：`/screen 北向连续加仓 + 业绩超预期`

## 工作流
1. 理解用户意图（量化 / 主题 / 事件）
2. 设定参数
3. 触发 `idea-generation` skill
4. 输出标的池 + 标签 + 业务一句话

## 示例
- `/screen 沪深300 内 PE-TTM<20 且 ROE>15%`
- `/screen 中特估 高股息`
- `/screen 近 5 日机构席位连续买入`
