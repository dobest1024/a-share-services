# 反洗钱（人行口径）

## 法规依据

- 《反洗钱法》（2007）+ 修订（征求意见稿 2024）
- 《金融机构客户身份识别和客户身份资料及交易记录保存管理办法》（人行 / 银保监 / 证监）
- 《金融机构大额交易和可疑交易报告管理办法》（人行）

## 三大核心义务

### 1. 客户身份识别（KYC）

| 阶段 | 内容 |
|---|---|
| 初次建立 | 身份证件、姓名、地址、职业、联系方式、风险评估 |
| 持续识别 | 资料更新、交易模式监控 |
| 强化识别 | 高风险客户（PEP / 特定行业 / 资金来源不清） |

### 2. 客户资料 / 交易记录保存
- 客户资料：业务关系结束后 **10 年**
- 交易记录：**10 年**

### 3. 报告义务
- **大额交易报告（LCT）**：达到金额阈值即报，无主观判断
  - 自然人单笔现金 ≥ 5 万元
  - 法人单笔境内非现金 ≥ 200 万元
  - 跨境汇款相关阈值
- **可疑交易报告（STR）**：基于专业判断
  - 客户行为异常
  - 交易特征异常
  - 资金来源 / 用途异常

## 证券基金行业典型可疑模式

1. **短期大额申赎**：账户开立后短期内大额申购又赎回
2. **资金通道**：账户作为他人转账通道（多源资金进、单一去向）
3. **价格操纵嫌疑**：单一账户对小盘股集中买入 / 拉抬涨停
4. **场内场外联动**：场内异动 + 场外可疑资金
5. **大宗承接折扣**：异常折扣大宗接盘可能涉及利益输送
6. **关联账户对倒**：多账户在同标的之间相互交易
7. **离岸资金 / 境外股东**：资金来源未充分披露

## A 股特殊场景

### 北向交易反洗钱
- 北向资金通过香港中央结算账户，穿透到底层投资者难度大
- 加强对北向异常买卖的监测
- 重大账户关联性核查

### 沪深港通账户
- 跨境账户需关联境外身份信息
- 港币 / 人民币兑换路径核查

### 私募 LP 穿透
- 私募基金 LP 必须穿透到自然人 / 上市公司 / 政府背景机构
- 多层嵌套 SPV 警惕

## 工作流

1. 输入：客户 / 账户活动数据
2. 自动化规则筛查（金额阈值 / 模式匹配）
3. 人工复核（可疑交易判断）
4. 报告（LCT 自动报送、STR 经合规判断）
5. 留痕
