# 多因子合成

## 合成方法

### 1. 等权合成
最简单：所有因子标准化后求平均。
适用：因子已通过单因子检验，IC 接近、相关性低。

### 2. IC 加权
权重 = 历史 IC 均值 / 历史 IC 标准差（即每个因子的 IR）。
适用：因子之间 IC 差异明显。

### 3. 最大化 IR
求解最优权重使组合 IR 最大化（基于因子 IC 协方差矩阵）：
```
w = Σ⁻¹ × IC_mean
```
其中 Σ 是因子 IC 之间的协方差矩阵。
适用：因子较多、相关性高时。

### 4. Barra 多因子（行业 + 风格）
把行业作为风格因子之一，整体回归后取超额。
适用：构建对冲组合时。

## 因子相关性管理

合成前必须检查：
- 因子两两 Pearson 相关性
- 相关 > 0.7 的因子保留代表性强的，剔除冗余
- 主成分分析（PCA）降维（适用因子很多时）

## A 股常见组合范式

### 价值 + 质量
代表组合：低 PB + 高 ROE + 稳现金流
- 持仓特点：大盘蓝筹偏多
- 风格：偏防御
- 中长期超额：稳定但弹性小

### 成长 + 动量
代表组合：高收入增速 + 6-1 动量 + 业绩预告超预期
- 持仓特点：中小盘成长股
- 风格：进攻
- 风险：风格切换时回撤大

### 多策略均衡
价值（25%）+ 质量（25%）+ 动量（25%）+ 反转（25%）
- 用于稳健配置
- 减少单一风格暴露

### 高股息（中特估）专项
低 PB + 高股息率 + 高 ROE + 央国企标签
- A 股 2023+ 受政策催化
- 跟踪：央企估值修复指数

## 工作流

1. 输入：因子列表（已通过单因子检验）+ 合成方法
2. 计算因子相关性矩阵 → 剔除冗余
3. 按方法求权重
4. 合成综合因子值
5. 全因子链路回测（单因子 → 组合 → 实盘模拟）
6. 输出综合因子的 IC / IR / 分组 / 多空收益
