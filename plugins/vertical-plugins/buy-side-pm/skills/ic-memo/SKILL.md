# 投资决策备忘录（IC Memo）

## 适用场景

任何**新建 / 加仓 / 减仓 / 退出**仓位前都需要 IC Memo（投资决策备忘录）。规模大的私募 / 公募有正式投决会，规模小的至少需要内部留痕。

## 标准结构

```
# IC Memo - {标的} - {决策类型} - {YYYY-MM-DD}

## 决策摘要（首页）
- 决策：{新建 / 加仓 / 减仓 / 退出}
- 标的：{公司}（{代码}）
- 规模：当前持仓 X% → 目标 Y%
- 期望持有：{短期 / 中期 / 长期}
- 关键论点（3 条）
- 关键风险（3 条）

## 投资逻辑
### 1. 业务理解
- 公司主业 / 商业模式 / 竞争优势
- 行业空间与公司位置

### 2. 财务质量
- 触发 `financial-quality-check` skill
- 关键指标：ROE、现金流、负债结构、商誉

### 3. 估值
- 当前估值水位（自身分位 + 行业分位）
- 主估值方法的结论
- 多方法交叉验证

### 4. 催化剂
- 已知 / 预期催化剂（财报、政策、产品、订单、重组）
- 时间表

## 论点细化
对前页 3 条核心论点逐一展开：
- 论点 1：{陈述}
  - 当前证据
  - 验证指标
  - 反证条件
- 论点 2
- 论点 3

## 风险分析
- 业务风险
- 财务风险
- 流动性风险
- 政策 / 行业风险
- 竞争 / 替代风险

## 仓位管理
- 建仓节奏：一次性 / 分批
- 加仓 / 减仓的触发条件（不出绝对价位，出"相对沪深 300 跑输 X%" 或 "估值进入 X 分位以上"等）
- 止损 / 止盈框架

## 与现有组合的协同
- 行业 / 风格暴露变化
- 与现有持仓的相关性
- 流动性影响

## 同业对比
- 同行业可比标的
- 为什么选这只而不是其他

## 卖方观点综述
- 一致预期 / 评级分布
- 与本次决策的差异

## 决策建议
- 强烈推荐 / 推荐 / 中性 / 保守（这是 IC 内部用语）
- 决议要点
- 后续监控清单

## ⚠️ 合规
- 关联交易 / 利益冲突披露
- 信息来源（公开披露 / 调研 / 卖方报告 / 其他）
- 决策权限审批
```

## A 股买方特殊点

### 风格漂移
- IC Memo 需说明决策与基金风格的一致性
- 偏离基金宣称风格的决策需要特别说明

### 流动性约束
- 个股流动性测试：自由流通市值 / 日均成交 / 持仓占自由流通比
- 大规模买入对市场冲击的预估

### T+1 与新规
- 持仓周期短的决策需要说明 T+1 影响
- 减持新规对大股东 / 高管持有的限制

### 北向参考
- 北向占比是参考但不是依据
- 北向流出风险测算

## 输出

每份 IC Memo 长度因决策重要性而异，一般 3-10 页：
- 小调仓：1-2 页
- 新建大仓位：5-10 页
- 重要新建（> 5% 仓位）：10+ 页

## 留痕

- 版本号管理（决策前、决策后修订）
- 审批人 / 决策日期
- 后续每季度 follow-up（论点是否兑现）
