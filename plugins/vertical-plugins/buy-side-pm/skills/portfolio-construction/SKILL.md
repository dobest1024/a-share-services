# 组合构建

## 权重方法

### 1. 等权
最简单：N 只标的，每只 1/N。
适合：标的池筛选信号强、流动性接近时。

### 2. 信号加权
权重 ∝ 信号强度（如综合因子 z-score 或卖方覆盖评级）。
适合：能量化信号差异时。

### 3. 风险平价（Risk Parity）
每只标的对组合风险贡献相同。
- 计算每只波动率
- 权重 ∝ 1/波动率
适合：避免高波股主导组合。

### 4. 均值方差（Markowitz）
最大化预期收益 / 波动率（即 Sharpe）。
- 需要预期收益向量（来自因子模型或卖方目标价）
- 需要协方差矩阵（用历史 + 收缩）
- 加约束（持仓上下限 / 行业暴露）

### 5. 行业中性
让组合行业暴露接近基准（如沪深 300 行业权重）。
适合：以指数增强为目标的组合。

## A 股组合构建约束

### 公募基金约束
- 单只 ≤ 10%（监管）
- 同一基金管理人持有同一证券 ≤ 10%
- 持仓股票 ≥ 80%（混合型）/ 60%（灵活配置型）
- 现金 + 流动资产 ≥ 5%
- 港股通占比限制（视产品而定）

### 私募基金约束
- 通常更灵活，但合同约定的杠杆 / 集中度限制要遵守
- 单只持仓上限有产品级约定

### 行业暴露
- 单行业不超过 30%
- 非基准成份股 ≤ X%

### 风格暴露
- Barra 风格暴露分解
- 与基金宣称风格一致性

## 工作流

1. 输入：标的池 + 基金类型 / 规模 / 约束
2. 选择权重方法
3. 构建优化问题
4. 求解（cvxpy / scipy）
5. 后处理：行业 / 集中度 / 流动性最终校验
6. 输出"权重表 + 风险报告"
