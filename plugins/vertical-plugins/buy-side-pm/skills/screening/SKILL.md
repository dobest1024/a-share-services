# 买方选股

## 与卖方 idea-generation 的差异

| | 卖方 idea-generation | 买方 screening |
|---|---|---|
| 目的 | 提供研究覆盖候选 | 直接生成投资标的池 |
| 约束 | 弱（输出大池）| 强（市值 / 流动性 / 行业 / 风格暴露上限）|
| 后置 | 等卖方研究员深度跟进 | 直接接组合构建流程 |
| 输出 | 50-100 标的池 | 30-50 标的池 + 详细标签 |

## 买方约束

### 流动性约束
- 自由流通市值 > X 亿（视基金规模而定，公募一般 > 50 亿）
- 日均成交 > X 千万
- 北向持股占比警戒（避免单日大额流出）

### 持仓集中度约束
- 单只持仓不超过 X%（监管 + 内控）
- 同行业持仓不超过 X%
- 同板块（创业板 / 科创板）不超过 X%

### 风格暴露
- 与基金风格一致（成长基金不应大量低估值价值股）
- 风格漂移监控

### 排除清单
- 行业排除（ESG / 个人偏好）
- 公司治理黑名单（频繁监管函 / 立案历史）
- 一致行动人变更频繁
- 财务舞弊嫌疑

## 工作流

1. 输入：基金类型 / 规模 / 风格 / 当前持仓约束
2. 多因子筛选（同 `quant-research/factor-library`）
3. 流动性 + 集中度过滤
4. 排除清单过滤
5. 业务分类标签（产业链位置 / 主题归属）
6. 输出"标的池 + 标签 + 业务一句话"

## 输出

```
## 买方选股池 - {筛选意图} - {YYYY-MM-DD}

### 筛选条件
- 基金类型：{...}
- 规模：{...}
- 风格：{价值 / 成长 / 均衡 / 高股息}
- 流动性：自由流通 > X 亿
- 行业限制：{...}
- 排除清单已应用

### 标的池（N 只）
| 代码 | 公司 | 行业 | 市值 | 风格标签 | 业务一句话 | 关注度 |

### 风格分布
- 大盘 / 中盘 / 小盘
- 价值 / 成长 / 均衡
- 行业分布

### 跟踪安排
- 调研优先级
- 财务质量复核需求
- 卖方研报参考

### 风险提示
- 标的池基于历史数据筛选，需配合业务尽调
- 不构成最终投资决策
```
