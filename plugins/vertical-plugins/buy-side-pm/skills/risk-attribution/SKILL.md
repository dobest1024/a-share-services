# 风险归因

## Barra 框架（A 股本地化）

### 风险模型结构
```
R_i = Σ X_ik × f_k + u_i
```
- R_i：个股收益
- X_ik：个股 i 在因子 k 上的暴露
- f_k：因子 k 的收益
- u_i：特质收益（不能用因子解释的部分）

### A 股风格因子（CNE6 简化版）

| 风格因子 | 简化定义 |
|---|---|
| Size | log(市值) |
| Beta | 个股 vs 中证 800 的 Beta |
| Momentum | 6-1 动量 |
| Reversal | 短期反转（1 月反向）|
| Volatility | 60 日波动率 |
| Liquidity | 月均换手率 |
| Value | BP / EP 综合 |
| Quality | ROE + 杠杆 + 盈利稳定性 |
| Growth | 收入 / 净利润增速 |
| Earnings Yield | EP-TTM |
| Dividend Yield | TTM 股息率 |

### 行业因子
- 申万一级 31 个哑变量
- 或中信一级 30 个

### 个股层面
对组合 Top 10 持仓：每只的收益来自哪里。

## A 股归因注意

### 因子选择
- 不要直接套用美股 Barra，A 股 Size 因子主导性强
- 北向占比 / 两融占比 / 龙虎榜活跃度等本土因子可加进风险模型

### 周期性
- 价值因子在 A 股周期性强，归因结论与时间区间敏感
- 至少做近 1 / 3 / 12 月三种区间

### 与策略宣称的一致性
- 价值策略基金风格暴露应在 Value > 0
- 成长策略基金 Growth > 0
- 漂移触发预警

## 输出结构

```
## 组合风险归因 - {组合名} - {区间}

### 收益归因（区间累计）
| 来源 | 收益贡献 | 占比 |
（多层级表格）

### 风险归因
（多层级表格）

### 风格暴露快照
| 风格 | 当前暴露 | 历史均值 | 偏离 |

### 行业暴露
| 申万一级 | 组合 | 基准 | 偏离 |

### Top 10 持仓贡献
| 标的 | 权重 | 区间收益 | 收益贡献 |

### 异常识别
- 暴露漂移（与历史均值偏离 > 1σ）
- 单只贡献过大（> 整体 20%）
- 特质收益占比异常（> 50% 表示因子模型解释力不足）

### 建议
- 暴露调整
- 集中度调整
- 跟踪误差控制
```
