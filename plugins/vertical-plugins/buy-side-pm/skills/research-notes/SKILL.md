# 调研纪要整理

## 调研类型

### 1. 上市公司调研
- 现场调研（工厂 / 总部）
- 电话会议
- 业绩说明会
- 反路演

### 2. 专家访谈
- 行业专家
- 公司前员工
- 供应商 / 客户
- 渠道商
- 行业协会

### 3. 渠道调研
- 经销商
- 终端门店

## 标准纪要结构

```
# {公司}/{行业} 调研纪要

## 基本信息
- 时间：YYYY-MM-DD
- 形式：{现场 / 电话 / 视频}
- 公司接待：{姓名 / 职位}
- 调研机构：{...}
- 调研要点（10-20 字摘要）

## Q&A 核心要点
按主题分类（避免按时间顺序）：

### 业务进展
- Q：...
- A：...（关键数据加粗）

### 财务展望
- Q：...
- A：...

### 行业景气
- Q：...
- A：...

### 竞争格局
- Q：...
- A：...

### 政策影响
- Q：...
- A：...

## 关键数据
（公司披露的非公开数据，标注是否为预期 / 实际）
- 指标 1：数值，时间
- ...

## 风险点
- 公司主动提示的风险
- 调研中察觉的风险（管理层闪烁、数据不一致、回避问题）

## 与公开披露差异
- 调研内容与最近财报 / 公告的差异
- 如有差异 → 警惕 ⚠️

## 跟踪建议
- 验证指标（下季报关注什么）
- 进一步访谈对象
- 与其他公司 / 专家交叉验证

## ⚠️ 合规
- 公司是否做信息披露承诺？
- 是否涉及非公开重大信息（如有需走合规流程，禁止用于交易）
```

## A 股合规雷区

### 信息披露公平性
- 公司应同时向所有投资者披露重要信息
- "选择性披露"违规：未公告的业绩 / 重大事项不应只告知部分机构
- 若获得非公开重大信息 → 立即停止交易、上报合规

### 调研记录与披露
- 接待机构调研的公告（上市公司义务）
- 调研纪要记录留底

### 内幕信息
- 重大资产重组、业绩重大变化、控制权变更、立案调查等
- 任何接触到内幕信息的方，在公开披露前不得交易

## 工作流

1. 输入：调研原始记录（文字稿 / 录音转写）
2. 按主题分类问题
3. 提取关键数据
4. 标注风险点
5. 与公开披露对照
6. 输出标准纪要

## 注意

- 纪要中**所有数据须标"公司披露 / 调研得知 / 自行估算"**
- 不得直接引用为对外研报数据，须二次确认
