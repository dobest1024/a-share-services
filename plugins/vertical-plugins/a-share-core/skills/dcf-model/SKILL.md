# A 股 DCF 估值

## 适用范围

适用：现金流可预测的非金融、非周期顶部公司（白电、酒、医疗器械、消费、稳态成长 TMT）。
不适用：银行/保险（用 DDM 或 P/B-ROE）、周期股顶部、亏损/早期成长（用 PS / EV/Sales）。

## 关键参数（中国语境）

### 无风险利率
- 10 年期国债收益率（取近 30 日均值，避免单日波动）
- 数据源：`akshare-mcp.macro.bond_yield_10y`

### 市场风险溢价（ERP）
- A 股本地化估计：**6-8%**（学术常用 6.5%，机构常用 7%）
- 不要直接搬美股的 5-6%

### 贝塔
- 优先用申万二级行业贝塔（个股贝塔噪声大）
- 数据源：`tushare-mcp.factor.beta`
- 周期股贝塔可能 > 1.3，稳定消费 0.6-0.9

### WACC
```
WACC = E/V × Re + D/V × Rd × (1 - 税率)
Re = Rf + β × ERP
税率 = 25%（标准）/ 15%（高新技术企业、西部大开发）
```

### 永续增长率
- A 股普遍取 **2-3%**（接近长期 GDP 增速）
- 超过 4% 需要特别论证
- 永续 = 名义增长，不要混淆实际增长

### 自由现金流口径
```
FCFF = 净利润 + 折旧摊销 - 资本支出 - 营运资金增加 + 利息费用 × (1 - 税率)
```

## A 股特有调整

1. **政府补助还原**：把"其他收益"中持续性补贴算入经常性 FCF；一次性补贴剔除
2. **少数股东权益**：合并报表里有大量少数股东时，估值要扣减
3. **永续债**：从权益移回负债，重算 D/V
4. **限制性现金**：受限的货币资金不算自由现金（如保证金、专款专用）

## 敏感性表

固定要做：WACC × 永续增长率 5x5 矩阵
- WACC 步长 0.5%
- 永续增长率步长 0.5%
- 输出每个组合的每股价值

## 输出结构

```

### 参数
- Rf = X%（10 年国债 30 日均值）
- ERP = 7%（A 股本地估计）

- WACC = X.XX%
- 永续增长率 g = 2.5%
- 预测期 = 5 年 + 永续

### 显性期假设（5 年）
| 年份 | 收入增速 | 营业利润率 | 资本支出/收入 | 营运资金/收入 |
| ... |

### 估值结果
- 企业价值 EV = XX 亿
- 净债务 = XX 亿
- 少数股东权益 = XX 亿
- 股权价值 = XX 亿
- 每股价值参考 = X.XX 元 （非目标价）

### 敏感性矩阵
（WACC 行 × g 列）

### 关键风险
- 显性期增速假设的依据
- 永续期可比性
- 资本支出周期判断
```
