# CAS 三表预测模型

## 与美股三表的关键差异

1. **利润表科目顺序**：A 股先列"营业总收入 / 营业收入"，再扣减"营业总成本"（含主营、税金、研发、销售、管理等）
2. **其他收益 / 资产处置收益**：政府补助和处置收益在 A 股是独立科目，营业利润之内
3. **少数股东损益**：合并报表里需要单独列示
4. **永续债利息**：分配给永续债持有者的金额计入未分配利润分配，不进利润表

## 预测顺序

### Step 1 收入（驱动一切）
分业务/分产品/分地区拆分，避免一刀切。
- 量 × 价 拆分（适用于消费、能源）
- 客户结构（适用于 B2B、订单驱动）
- 渠道结构（适用于线上 / 线下混合）

### Step 2 利润表
- 毛利率 = 1 - 营业成本/营业收入。看历史区间 + 行业可比
- 三项费用率（销售/管理/研发）通常与收入规模相关，规模效应导致下降
- 研发资本化率（高科技公司重点）
- 财务费用：与有息负债联动，预测时连资产负债表

### Step 3 资产负债表（关键勾稽）
- 营运资金（应收 / 存货 / 应付）按周转天数推
- 固定资产：期初 + 资本开支 - 折旧
- 长期借款 + 短期借款：根据资金缺口推

### Step 4 现金流量表
**核心勾稽**：
- 经营性现金流 ≈ 净利润 + 折旧摊销 - 营运资金增量
- 投资性现金流 = -(资本开支 + 长期股权投资增量)
- 筹资性现金流 = 借款净增加 - 分红 - 回购 + 增发

## A 股特有处理

### 政府补助预测
- 与日常活动相关：作为收入的某个百分比（通常 1-3%，半导体/新能源可能 5%+）
- 一次性：不预测，只在已公告时记入

### 商誉
- 历史并购形成的商誉默认不减值，但要标"减值风险敞口"
- 业绩承诺期内 / 期满 1-2 年的标的要重点提示

### 永续债
- 已有的：单独列示，不算负债，但利息按非控制权益分红处理
- 新增的：估值时还原回负债

### 限售股解禁
- 解禁前后流通股本变化要在 EPS 中体现
- 总股本预测要考虑可转债转股、股权激励行权

## 输出结构

简化版（适合 agent 一次性输出）：

```
## 三表预测（{公司}）

### 关键假设
- 收入 CAGR 5Y = X%（基于 ... 驱动因素）
- 毛利率 = X%（vs 历史 Y%）
- 三项费用率合计 = X%
- 资本开支强度 = X%
- 营运资金/收入 = X%

### 预测三表（简化）
| | 2024A | 2025E | 2026E | 2027E |
| 收入 | ... |
| 营业利润 | ... |
| 归母净利润 | ... |
| 资产负债率 | ... |
| 经营性现金流 | ... |
| 自由现金流 | ... |

### 关键勾稽校验
- 现金流量表期末现金 = 资产负债表货币资金 ✓
- 净利润 → 未分配利润增加 ✓
- 资本开支 → 固定资产增加（扣折旧后） ✓
```
