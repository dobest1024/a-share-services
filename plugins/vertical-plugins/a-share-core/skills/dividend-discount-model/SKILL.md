# 股息贴现模型 / 高股息策略

## 核心公式（戈登模型）

```
P = D₁ / (r - g)
```

适用前提：股息持续派发、增速稳定、r > g。

## A 股高股息策略适用领域

| 行业 | 典型代表 | 股息率（近 5 年中枢）|
|---|---|---|
| 煤炭 | 中国神华、陕西煤业 | 6-10% |
| 银行 | 工行、建行、招行 | 4-6% |
| 电力 | 长江电力、华能国际 | 3-5% |
| 高速公路 | 宁沪高速、招商公路 | 5-7% |
| 运营商 | 中国移动、中国电信 | 4-6% |
| 石油石化 | 中石油、中石化 | 5-8% |
| 公用事业 | 三峡能源、川投能源 | 3-5% |

## 关键指标

- **股息率**：每股股息 ÷ 当前股价（用 TTM 股息更稳）
- **分红率**：股利 ÷ 净利润，A 股优质央国企普遍 30-50%
- **股息支付的可持续性**：自由现金流 ≥ 派息额 + 维持性资本开支

## 中特估语境

2022 末起监管引导央国企估值修复，**中特估**主题让国有大型央企股息率提升路径变成：
1. 提分红比例（政策推动 → 直接抬升股息率）
2. 利润稳定 → 股息可持续
3. 估值修复（PB 从 0.4 修到 0.6-0.8）

跟踪信号：央企 KPI 考核口径变化、国务院国资委文件、央企集团表态。

## 输出结构

```
## 高股息分析（{公司}）

### 股息历史
| 年份 | 每股股息 | 分红率 | 股息率（按当年均价） |

### 当前指标
- TTM 股息率 = X.X%
- 5 年平均分红率 = X%
- 自由现金流覆盖股息倍数 = X.X 倍

### 可持续性
- 经营性现金流 / 派息额（应 > 1.5）
- 资本开支强度（是否扩张期吞噬现金）
- 资产负债率水平
- 控股股东对分红的诉求（央企 / 民企家族 / 财务投资人）

### 参考观点
- 当前股息率 vs 历史分位
- 与 10 年国债（无风险利率）的息差
- 类债属性强弱

### 风险
- 行业景气下行 → 利润下滑 → 股息缩水
- 政策风险（资源税、电价、利率市场化）
- 分红率回调
```
