# PB-ROE 估值框架

## 核心公式

```
PB ≈ (ROE - g) / (r - g)
```

其中：
- ROE：可持续股本回报率
- g：长期增长率
- r：股权资本成本（用 CAPM 估计）

简化判断：**PB 应该与 ROE 同方向。** ROE 高的公司应有更高 PB；偏离时是机会或风险。

## 适用范围

| 行业 | 用 PB-ROE 的理由 |
|---|---|
| 银行 | PE 受拨备调节大；PB 与 ROE 关系稳定 |
| 保险 | EV（内含价值）是更精细口径，但 PB-ROE 仍是粗筛 |
| 券商 | 周期性强，PE 失真，PB 提供下限 |
| 钢铁、煤炭、有色 | 周期顶部 PE 极低、底部 PE 极高，PB 更稳定 |
| 地产开发 | 存货占比大、利润释放滞后 |
| 公用事业 | 资产驱动 |

## 操作步骤

1. **横截面对比**：散点图，X 轴 ROE，Y 轴 PB，画申万二级所有公司
2. **拟合线**：用 OLS 拟合 PB = a + b × ROE
3. **位置判断**：
   - 拟合线**之上** → 相对溢价（除非有特殊原因如增速）
   - 拟合线**之下** → 相对折价（可能机会 / 也可能基本面恶化）
4. **时间序列**：标的自身近 5 年 PB-ROE 散点

## A 股银行专用

- ROE 中枢 10-15%（国有大行）/ 12-18%（股份制 + 城商行优秀者）
- PB 中枢 0.5-0.7（破净是常态，过去 10 年）
- 重点变量：拨备覆盖率 / 不良率 / 净息差 NIM / 中收占比
- 中特估行情下，红利策略让国有大行 PB 从 0.4 修复到 0.6

## 输出结构

```
## PB-ROE 分析（{公司} vs 申万二级 {行业}）

### 标的位置
- 标的 ROE-TTM = X%；PB = X.XX
- 行业中位 ROE = X%；PB = X.XX
- 拟合线预测 PB（基于本公司 ROE）= X.XX
- 偏离度 = +/- X%

### 时间序列
- 近 5 年 PB-ROE 散点（标的）
- 当前位置 vs 历史中枢

### 参考观点
- 折价/溢价的可能原因（业绩稳定性、资产质量、增长预期）
- 关注后续 ROE 中枢是否能维持

### 风险提示
- ROE 可持续性（一次性损益 / 杠杆放大）
- 行业周期位置
```
