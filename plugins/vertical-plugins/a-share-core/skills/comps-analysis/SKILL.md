# A 股可比公司分析

## 估值指标选择

| 行业类型 | 主用 | 备用 |
|---|---|---|
| 消费、医药、TMT 成长股 | PE-TTM / PEG | PS |
| 周期股（钢铁、煤炭、化工、地产）| PB | PE-中枢 |
| 金融（银行、保险）| PB-ROE | 股息率 |
| 公用事业、高速公路 | 股息率 / EV/EBITDA | PE |
| 亏损 / 早期成长 | PS | EV/Sales |
| 资源型 | EV/EBITDA | PB |

**禁忌**：对周期股顶部不要用低 PE 当便宜（盈利高峰错觉）；对亏损股不要用 PE。

## 同行筛选

1. 按申万二级行业初筛（一般 5-15 家）
2. 二级太宽时下沉到申万三级
3. 排除条件：
   - ST / *ST（基本面不可比）
   - 上市不足 1 年（数据不全）
   - 市值差 10 倍以上（除非业务真的可比）
   - 主营业务实质偏离（如"医药制造"二级里的医美 vs 仿制药）

## 估值矩阵输出

横坐标：估值指标；纵坐标：公司列表。固定列：
- 公司 / 代码 / 申万三级
- 市值（亿元）
- 收入 TTM / 同比
- 归母净利润 TTM / 同比
- ROE-TTM
- PE-TTM / PB / PS / 股息率
- PE 近 5 年分位 / PB 近 5 年分位

## A 股特有提醒

- **流通市值优先**：A 股有大量限售股，比较时用自由流通市值更接近"可定价市值"
- **创业板/科创板溢价**：同业务的创业板公司估值常高 30-50%，是否调整看口径
- **次新股**：上市 1 年内估值通常虚高，避免拉入均值
- **北向资金重仓股**：核心资产估值有"陆股通溢价"，与非重仓股对比需说明
- **大小盘风格切换**：2021 后小盘超额明显，对比时同时给出 沪深300 / 中证500 / 中证1000 子集中位数
