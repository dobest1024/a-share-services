# 监管函与问询函解读

## 监管文书类型（严重性递增）

| 文书 | 发文方 | 性质 | 严重性 |
|---|---|---|---|
| 关注函 | 交易所（沪深 / 北交所）| 关注业务事项 | ★ |
| 问询函 | 交易所 | 要求公司解释 | ★★ |
| 年报问询函 | 交易所 | 年报披露后例行 | ★★（普遍）|
| 重组问询函 | 交易所 | 针对重大资产重组 | ★★ |
| 监管函 | 交易所 / 证监局 | 已经认定违规，行政监管 | ★★★ |
| 警示函 | 证监局 | 行政监管措施 | ★★★ |
| 立案告知书 | 证监会 | 启动立案调查 | ★★★★ |
| 处罚决定 | 证监会 | 调查后处罚 | ★★★★★ |
| 强制退市决定 | 交易所 | 极端情形 | ★★★★★ |

## 内容关键字判断

### 业务/财务类问询（中性）
- "请补充说明" + 商誉 / 应收账款 / 存货 / 关联交易 / 业绩波动
- 年报后常规问询，多数公司能完整回复，事件本身严重性不大

### 警示类（关注）
- "可能违规"、"信息披露不规范"
- 实控人 / 大股东行为问询
- 业绩预告与实际差异过大的问询

### 严重类（高风险）
- "涉嫌信息披露违法违规" → 立案调查信号
- "财务造假" / "虚增收入" / "占用资金" → 退市风险
- "未及时披露"重大事项 → 内部治理问题

## 历史复盘模式

| 公司情形 | 后续走势 |
|---|---|
| 商誉 / 应收常规问询 + 充分回复 | 短期下跌 5-10%，2 周内修复 |
| 监管函 + 信息披露违规 | 短期下跌 15-25%，长期看后续 |
| 立案告知 + 财务造假嫌疑 | 短期跌停 + 长期 ST / 退市 |
| 实控人立案 + 公司经营受影响 | 股价腰斩级别 |

⚠️ 历史规律仅供参考，每个案例需独立判断。

## 工作流

1. 拿到监管文书 PDF（`cninfo-mcp.filing_text(url)`）
2. 提取：发文方 / 文书类型 / 主要质询点 / 回复期限
3. 与历史同类型问询对比严重性
4. 评估对基本面与股价的潜在影响
5. 输出"事件等级 + 关注点 + 后续观察清单"

## 输出结构

```
## 监管文书解读 - {标的} {日期}

### 文书基本信息
- 类型：{文书类型}
- 发文方：{发文方}
- 标题：{标题}
- 严重性等级：★ ~ ★★★★★

### 核心质询点
- 1. ...（公告原文引用 + 自然语言解读）
- 2. ...
- 3. ...

### 公司回复要求
- 回复期限：{X 个工作日}
- 是否需要中介机构核查（会计师 / 律师 / 独立财务顾问）

### 严重性评估
- 与历史同类型对比
- 对基本面的影响
- 对股价的潜在影响（短期 / 中期）

### 后续观察清单
- 关注公司回复函发布
- 关注是否升级（关注函 → 问询函 → 监管函 → 立案）
- 关注交易异常（涨跌停 / 大宗 / 龙虎榜）

### ⚠️ 重要提示
- 监管文书严重影响股价，但具体走势取决于回复质量与后续处置
- 不构成任何投资建议
```
