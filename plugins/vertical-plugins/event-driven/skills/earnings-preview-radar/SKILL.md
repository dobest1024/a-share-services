# 业绩预告 / 业绩快报扫描

## A 股两种"提前披露"

### 业绩预告（强制 + 自愿）
强制触发：
- 净利润同比 ±50% 或更多
- 扭亏为盈 / 首次亏损
- 净利润上下限差异较大

披露时点：
- 年报：1 月底前（强制触发）
- 半年报：7 月中
- 季报：4 月中 / 10 月中

### 业绩快报（自愿）
- 早于正式财报披露
- 仅披露主要财务数据（无完整附注）
- 通常出现在年报前 1-2 个月

## 解读维度

### 1. 与预期对比
- 卖方一致预期（Wind / Choice 一致预期）
- 同比 / 环比
- "略超预期" / "明显超预期" / "符合" / "低于" / "大幅低于"

### 2. 超预期类型
| 超预期来源 | 持续性 |
|---|---|
| 主业量价齐升 | 高 |
| 行业景气拐点 | 高 |
| 新业务突破 | 中-高 |
| 成本下降（原材料 / 汇率）| 中（依赖外部）|
| 政府补助 / 投资收益 | 低（一次性）|
| 减值转回 | 低 |
| 会计估计变更 | 低 |

### 3. A 股财报披露季交易模式
- **预告期**（披露季前 2 周）：买入超预期标的，规避雷点
- **业绩雷扎堆期**（次年 1 月底、4 月底）：商誉减值 / 业绩大幅低于预期密集出现
- **业绩快报期**：业绩快报往往是"先报喜"，差业绩公司更倾向延迟披露

## 业绩雷信号

⚠️ 业绩预告披露前 5-10 天若出现以下信号，警惕：
- 公司高管减持
- 大额计提准备的公告
- 商誉减值测试公告
- 重要客户合同终止公告
- 财务总监 / 审计变更

## 工作流

1. 获取目标标的的业绩预告 / 快报（`akshare-mcp.fundamental.earnings_preview(code)`）
2. 与卖方一致预期对比（`tushare-mcp.consensus.estimate(code)`）
3. 拆分业绩超预期的来源
4. 输出"超预期等级 + 持续性 + 后续观察"

## 输出结构

```
## 业绩预告 / 快报 - {标的}

### 披露详情
- 类型：{业绩预告 / 业绩快报}
- 报告期：{2024Q4 / 2024H1 / 2024Q3}
- 披露日：{YYYY-MM-DD}

### 公司预告数据
- 归母净利润：{下限} ~ {上限} 万元
- 同比：{+X% ~ +Y%}
- 扣非归母：{...}

### 与预期对比
- Wind 一致预期：{X 万元}
- 偏离度：超预期 +X% / 符合 / 低于 -X%

### 超预期来源拆解（如适用）
- 主业贡献：X% （持续性高）
- 非经常性损益：X% （持续性低）
- 业务结构变化：...

### 历史对照
- 该公司过去 4 个季度的"预告 vs 实际"准确度
- 行业内其他公司同期表现

### 风险信号
- [Y/N] 高管近期减持
- [Y/N] 大额计提公告
- [Y/N] 审计 / 财务变更

### 关注要点
- 正式财报披露日
- 业绩说明会 / 投资者交流安排
```
