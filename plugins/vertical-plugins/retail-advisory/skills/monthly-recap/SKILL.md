# 客户月度市场综述

## 用途

发给投顾签约客户的月度市场综述，包括：
- 上月市场回顾
- 本月市场展望（不出具体操作建议）
- 客户持有产品的简要点评（不针对个股）
- 风险提示

## ⚠️ 合规要点

1. **所有内容必须经合规预审签字**
2. **不可点名具体个股**（除非该股是客户已持有且持牌人员确认）
3. **不可使用诱导性词汇**（"必涨"、"绝佳机会"、"火爆"）
4. **不可暗示择时能力**（"我们准确预测了上月调整"）
5. **风险提示必须显著、与正文同等显眼**
6. **数据来源全部标注**

## 标准结构

```
# {YYYY-MM} 市场综述
{机构名} | {投顾签字} | {合规审核签字}

## 上月回顾

### A 股表现
- 主要指数（沪深 300 / 中证 500 / 创业板指 / 科创 50）涨跌
- 行业表现（涨幅前 / 跌幅前 申万一级）
- 市场情绪（成交量 / 北向 / 两融 / 涨跌停潮）

### 主要事件
- 政策（不超过 3 条）
- 业绩披露（财报季的话）
- 海外（美联储 / 地缘）

### 板块亮点
- 强势板块的驱动
- 弱势板块的压制

## 本月展望

### 宏观环境
- 货币政策（央行近期表态 / LPR 走向）
- 财政政策
- 经济数据预期

### 市场层面
- 估值水位（中性 / 偏低 / 偏高）
- 风险偏好（怎么判断）
- 主要关注事件（本月即将发生）

### 行业关注（不出具体股票）
- 受益方向（基于政策 / 基本面）
- 风险方向

### 风格判断
- 价值 vs 成长
- 大盘 vs 小盘
- 周期 vs 防御

## 客户产品点评

### 产品 1（按客户持有的）
- 上月表现
- 持仓结构变化
- 基金经理动态
- 本月关注点

（同结构展开 2-3 个产品）

## 操作建议
**保留客户主动决策**

- 是否需要再平衡（按客户配置框架）
- 是否需要调整定投金额
- **不出"加仓 / 减仓某只产品"的具体建议**（须由持牌投顾会面后再具体讨论）

## 风险提示
（与正文同等显眼）

- 市场风险
- 政策风险
- 流动性风险
- 汇率风险（持有 QDII 的话）
- 个股 / 行业风险（持有主动股基 / 行业 ETF 的话）

## 数据来源
- Wind / Choice / 中国基金报 / 公司公告 / 央行官网 / 国家统计局

## 客户互动
- 月度回访预约方式
- 紧急联系渠道

---
{机构合规免责声明全文}
```

## 工作流

1. 输入：客户画像 + 客户持仓
2. 拉取月度市场数据
3. 按结构生成草稿
4. 客户产品段落按客户实际持仓生成
5. **草稿必须经合规预审 + 持牌投顾签字后才能发送**
6. 留痕（版本 / 审核 / 发送时间）

## 不可绕过的合规

- 即使 agent 生成完成，也只是草稿
- 投顾机构必须有内审流程
- 发送前必须签字
- 客户回复 / 反馈也需要留痕
