# 资产配置框架

## ⚠️ 合规前置

本 skill 只输出**资产大类配置框架**（股 / 债 / 商品 / 现金的比例区间），**不指定个股 / 个基的具体权重**。

未持有投资顾问业务资质（证监会发牌的"投资顾问"）的服务者不得提供具体投资建议。

## 配置框架五大类

| 大类 | 工具 | 风险 | 流动性 |
|---|---|---|---|
| 现金类 | 货币基金 / 银行存款 / T+0 理财 | 极低 | 高 |
| 固定收益 | 国债 / 政金债 / 信用债 / 债券基金 | 低-中 | 中-高 |
| 权益类 | A 股 / 港股 / 美股 / 主动股票基金 / 指数基金 | 高 | A 股 T+1 |
| 商品 | 黄金 / 原油 / 商品基金 | 中-高 | 中 |
| 另类 | REITs / 私募 / 衍生品 | 高 | 低 |

## 经典配置范式

### 60/40 经典配置
- 权益 60%
- 固定收益 40%
- 适合：长期、平衡型客户
- A 股语境：60% 中沪深 300 ETF + 中证 500 ETF + 优质主动股基；40% 中长债基

### 全天候（桥水原型）
- 权益 30%
- 长债 40%
- 短债 15%
- 黄金 7.5%
- 商品 7.5%
- 适合：稳健、追求低回撤

### 100 - 年龄
- 权益占比 = 100 - 年龄
- 例：30 岁 → 70% 权益
- 适合：随年龄推进自动降低风险

### A 股本地化模型
- 国内权益（A 股）40-60%
- 海外权益（QDII）5-15%
- 国内债券 20-40%
- 黄金（避险）3-10%
- 现金 5-15%

## 客户类型 → 配置范式映射

| 客户 | 推荐范式 |
|---|---|
| C1 谨慎 | 现金 30% + 固收 60% + 权益 10% |
| C2 稳健 | 现金 15% + 固收 60% + 权益 25% |
| C3 平衡 | 现金 10% + 固收 50% + 权益 35% + 黄金 5% |
| C4 积极 | 现金 5% + 固收 30% + 权益 55% + 黄金 5% + 商品 5% |
| C5 激进 | 现金 5% + 固收 15% + 权益 70% + 另类 10% |

## 工作流

1. 输入：客户画像（含适当性等级 / 投资目标 / 时间跨度）
2. 选择范式（或自定义）
3. 输出大类比例区间（不出具体产品）
4. 提示后续步骤（具体产品由持牌投顾决定）

## 输出

```
## 资产配置框架 - {客户编号} - {YYYY-MM-DD}

### 客户画像摘要
- 适当性等级：C{X}
- 投资目标：{...}
- 时间跨度：{X} 年
- 流动性需求：{...}

### 推荐范式
- 范式名：{...}
- 设计理念：{...}

### 大类配置比例（推荐区间）
| 大类 | 比例 | 说明 |
| 现金 | 5-15% | 紧急备用 + 流动性缓冲 |
| 固收 | 30-50% | 收益基础 + 抗波动 |
| 权益（含国内 / 海外）| 30-50% | 长期增长 |
| 黄金 / 商品 | 5-10% | 通胀对冲 |
| 另类 | 0-5% | 视客户认知 |

### 再平衡机制
- 触发条件：单一大类偏离目标区间 5pct
- 频率：季度或半年度
- 方式：定投修正 / 直接调仓

### ⚠️ 重要提示
- 本框架仅为大类配置建议
- **具体产品（个股 / 个基）的选择须由持牌投资顾问完成**
- 本输出不构成具体投资建议
- 客户应自行判断或寻求专业建议

### 后续流程
- 持牌投顾基于本框架做具体配置
- 客户签字确认配置方案
- 后续按再平衡机制定期复盘
```
