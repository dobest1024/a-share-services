# 定投策略

## 适用场景

定投适合：
- 长期投资（≥ 5 年）
- 月度有稳定收入
- 不具备择时能力
- 心理上承受不了一次性大额投入

不适合定投：
- 短期需用钱
- 已经处于估值历史高位（定投在高位入场效果差）
- 频繁波动的非主流标的

## 定投标的选择

### 优先级（按稳定性）
1. **宽基指数 ETF**：沪深 300、中证 500、中证 1000、上证 50、创业板指
2. **行业 ETF**：消费、医药、科技、新能源等
3. **主动股基**：业绩稳定、规模适中的明星基金
4. **指数增强基金**：跟踪宽基 + 量化超额

### 不建议
- 单一个股定投（风险集中）
- 主题 / 概念基金（波动大）
- QDII（汇率 + 海外双重风险，仅作小比例配置）

## 定投方式

### 1. 等额定投
固定金额，固定时间。**最简单**。

### 2. 估值加权定投
按当前估值水位调整定投金额：
- PE 历史分位 < 30%：加倍（200% × 基础金额）
- PE 历史分位 30-70%：正常（100%）
- PE 历史分位 > 70%：减半（50%）
- PE 历史分位 > 90%：暂停

### 3. 智能定投（移动平均线）
- 跌破 200 日线 X% 加倍
- 突破 200 日线 X% 减半

### 4. 目标止盈定投
设定目标收益率，达到后一次性赎回，重新开始。
- 适合宽基（如沪深 300 设 20-30% 止盈）
- 不适合主动股基（容易踏空持续行情）

## 关键参数

| 参数 | 推荐值 |
|---|---|
| 单期金额 | 月收入 10-20%（自动扣款） |
| 频率 | 月度（每月固定日）或周度 |
| 持续时间 | ≥ 3 年（最好 5-10 年）|
| 止盈方式 | 估值止盈（PE 分位 > 80%）/ 收益止盈 |
| 止损 | 通常不止损（定投本质是低位累积）|

## A 股定投特殊点

### 春节 / 长假效应
- 长假后通常有躁动 → 长假前后定投正常进行即可
- 不要主动避开

### 估值锚定
- 沪深 300 PE-TTM 10-15 倍是中枢
- 中证 500 PE-TTM 20-30 倍是中枢
- 历史分位是工具，不要僵化用

### 主动 vs 被动
- 长期 A 股主动基金跑赢宽基的能力强于美股
- 但选基难度大，新手建议先用宽基

## 工作流

1. 输入：客户画像 + 定投目标
2. 选择标的（宽基 / 行业 / 主动）
3. 选择定投方式（等额 / 估值加权 / 智能 / 目标止盈）
4. 测算预期收益 / 风险（基于历史回测）
5. 输出"定投方案 + 跟踪机制"

## 输出

```

### 客户与目标
- 时间跨度：{X} 年
- 目标收益：年化 X-Y%
- 月度可投金额：X 元

### 推荐方案
- 标的：{宽基 ETF / 主动基金 + 标的代码}
- 方式：{等额 / 估值加权 / 智能定投 / 目标止盈}
- 频率：{...}
- 单期金额：X 元

### 历史回测参考
（基于过去 10 年定投同一标的）
- 平均年化收益：X%
- 最大回撤：X%
- 收益正概率：X%

### 止盈机制
- 触发条件：{...}
- 操作方式：{...}

### 再投入
- 止盈后重新定投的标的与方式

### ⚠️ 风险与免责
- 历史回测不代表未来表现
- 长期定投仍有可能亏损（标的本身长期下跌）
- 本方案为框架，具体执行须由持牌投顾确认
```
