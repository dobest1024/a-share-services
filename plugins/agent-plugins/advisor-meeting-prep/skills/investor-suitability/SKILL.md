# 投资者适当性管理

> 与 `retail-advisory/suitability-check` 偏重交易前匹配不同，本 skill 偏重**合规运营**：分级制度、留痕、抽样复核、年度复评。

## 法规依据

- 《证券期货投资者适当性管理办法》（证监会 2017）
- 《公开募集证券投资基金销售机构监督管理办法》
- 《证券公司投资者适当性管理实施指引》

## 客户分级（C1-C5）

详见 `retail-advisory/client-profiling`。本 skill 关注**分级流程的合规执行**：

1. 客户填写风险评估问卷（最少 20 题）
2. 评估打分得到等级
3. 客户对评估结果签字确认
4. 录入客户资料库
5. 至少每 2 年复评一次

## 产品分级（R1-R5）

- 各家机构内部分级标准须与监管口径一致
- 新增产品上线前需做风险评级
- 产品评级变更需通知客户

## 销售流程留痕

- 双录（录音录像）
- 风险揭示书签字
- 适当性匹配记录
- 销售人员持证情况留底

## 抽样复核

每月 / 每季度抽样：
- 客户分级合理性
- 产品分级合理性
- 销售匹配
- 双录完整性
- 风险揭示书

抽样比例：合规要求至少 5%，关键人员 / 关键产品全样。

## 年度复评

每年至少一次：
- 老客户复评（特别是配置发生大变化的）
- 产品风险等级回顾
- 销售人员业绩与合规挂钩复核
