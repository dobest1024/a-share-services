# 投资者适当性匹配检查

## 监管框架

《证券期货投资者适当性管理办法》（证监会，2017）核心要求：
1. **了解客户**：完整客户画像
2. **了解产品**：风险等级
3. **匹配销售**：不得向不适合客户销售高风险产品
4. **录音录像 / 双录**：销售过程留痕
5. **风险揭示书**：签字确认

## 客户分级（C1-C5）

| 等级 | 描述 | 可购买产品 |
|---|---|---|
| C1 | 谨慎型 | R1（保本类） |
| C2 | 稳健型 | R1-R2 |
| C3 | 平衡型 | R1-R3 |
| C4 | 积极型 | R1-R4 |
| C5 | 激进型 | R1-R5（含衍生品 / 杠杆） |

## 产品风险分级（R1-R5）

| 等级 | 典型产品 |
|---|---|
| R1 | 国债、银行存款、货币基金 |
| R2 | 债券基金（纯债 / 二级债基）、保本理财 |
| R3 | 偏债混合基金、可转债 |
| R4 | 偏股 / 股票基金、A 股股票 |
| R5 | 期货、期权、私募、杠杆类、商品 |

## 投资品种 A 股特有提醒

| 品种 | 投资者门槛 |
|---|---|
| 主板股票 | 无 |
| 创业板 | 2 年交易经验 + 10 万资产 |
| 科创板 | 2 年交易经验 + 50 万资产 |
| 北交所 | 2 年 + 50 万 |
| 融资融券 | 50 万资产 + 半年交易 |
| 股指期货 | 50 万 + 知识测试 |
| 期权 | 50 万 + 三级测试 |
| 私募基金 | 100 万 + 个人 / 机构合格投资者 |

## 工作流

1. 输入：客户画像 + 候选产品
2. 检查客户风险等级与产品风险等级匹配
3. 检查投资品种门槛
4. 检查流动性匹配（投资期限 vs 客户期限要求）
5. 输出"匹配 / 不匹配 + 原因"
6. 不匹配时拒绝销售并留痕

## 输出

```
## 适当性匹配检查 - {客户编号} - {产品代码}

### 客户
- 风险等级：C{X}
- 可购买产品上限：R{Y}
- 关键约束：{投资品种门槛 / 流动性 / 投资期限}

### 产品
- 名称：{...}
- 风险等级：R{Z}
- 投资期限：{...}
- 投资品种：{...}

### 匹配结果
- 整体：✅ 匹配 / ❌ 不匹配
- 风险等级：✅ / ❌
- 投资品种门槛：✅ / ❌
- 流动性匹配：✅ / ❌
- 投资期限匹配：✅ / ❌

### 详细说明
（如不匹配，列出具体原因）

### 操作建议
- 匹配：可继续销售流程，需完成"双录"
- 不匹配：禁止销售，留痕记录

### ⚠️ 合规
- 本检查仅为合规辅助，最终判定由持牌人员确认
- 销售过程需"双录"（录音录像）
```
