# A 股因子库

## 核心因子分组（A 股本地化定义）

### 价值因子（Value）
| 因子 | 计算 | 数据源 |
|---|---|---|
| BP | 1 / PB-LF | tushare.fina |
| EP | 1 / PE-TTM | tushare.daily_basic |
| SP | 1 / PS-TTM | tushare.daily_basic |
| DivP | TTM 股息 / 价格 | tushare.dividend |
| FCFP | TTM FCF / 市值 | 自算 |

⚠️ A 股价值因子需排除：金融行业（特别 PB 失真）、ST 股、上市 < 1 年

### 质量因子（Quality）
| 因子 | 计算 |
|---|---|
| ROE | 归母净利 / 期初权益（最近年报）|
| ROA | 净利润 / 总资产 |
| 毛利率 | 1 - 营业成本/营业收入 |
| 资产周转率 | 收入 / 平均总资产 |
| 经营现金流/净利润 | OCF / 归母净利 |
| 杠杆率（反向）| 1 / (1 + 总负债/权益) |

### 动量因子（Momentum）
| 因子 | 计算 |
|---|---|
| MOM_12_1 | (P_t-21 / P_t-252) - 1，跳过最近一月 |
| MOM_6_1 | 6 月动量跳一月 |
| MOM_3M | 3 月动量 |
| 行业相对动量 | 个股 - 行业指数 |

A 股动量周期短：12-1 经典美股因子在 A 股不稳定，6-1 / 3M 更适用。

### 反转因子（Reversal）
| 因子 | 计算 |
|---|---|
| REV_1M | -1 × (近 21 日收益) |
| REV_5D | -1 × (近 5 日收益) |

A 股短期反转效应强（个人投资者占比高），1M 反转因子有效性显著。

### 成长因子（Growth）
| 因子 | 计算 |
|---|---|
| 收入 YoY | 单季收入同比 |
| 收入 QoQ | 单季收入环比 |
| 归母 YoY | 单季归母同比 |
| 收入 3Y CAGR | 3 年复合增速 |
| 业绩预告超预期 | (预告中值 - 一致预期) / 一致预期 |

### 规模因子（Size）
| 因子 | 计算 |
|---|---|
| LN_MV | log(总市值) |
| LN_FloatMV | log(自由流通市值) |

A 股小盘超额近年明显（2021 后），但有反转风险。

### 波动率因子（Volatility）
| 因子 | 计算 |
|---|---|
| RV_20 | 近 20 日收益率标准差 |
| RV_60 | 近 60 日 |
| Idio_Vol | 个股残差波动率（Fama-French 三因子回归后残差） |

低波因子在 A 股有效，但行情起来时受抑制。

### 流动性因子（Liquidity）
| 因子 | 计算 |
|---|---|
| Turnover_20 | 近 20 日均换手率 |
| Amihud | (\|R_t\| / Volume_t) 的均值 |
| Bid-Ask Spread | 买卖价差比 |

### 贝塔因子（Beta）
| 因子 | 计算 |
|---|---|
| Beta_252 | 个股 vs 沪深 300 近 252 日 OLS |
| Beta_60 | 短期 |

### 情绪 / 资金面因子（A 股特有）
| 因子 | 计算 |
|---|---|
| 北向占比 | 北向持股 / 自由流通股本 |
| 北向变化 | 北向持股 20 日变化率 |
| 两融占比 | 融资余额 / 流通市值 |
| 龙虎榜净买入 | 近 20 日龙虎榜净买入累计 |
| 主力净流入 | 大单 + 特大单净流入 |

## 因子使用注意

### 风格切换
单因子在 A 股的有效性会**断崖式切换**：
- 2017-2020：价值 + 大盘
- 2021-2023：成长 + 小盘
- 2024-：高股息 + 中特估
不要假设历史 IC 稳定。

### 行业中性化
A 股行业差异大，绝大多数因子计算前要做**行业中性化**：
- 用申万一级（31 个）作哑变量回归
- 取残差作为因子值

### 市值中性化
小盘因子主导收益，多数其他因子需中性化市值：
- 用 LN_MV 作 OLS 回归
- 取残差作为最终因子

### 缺失值
- 财务因子：按 fillna(行业中位) 处理
- 行情因子：剔除（不要 fillna）
