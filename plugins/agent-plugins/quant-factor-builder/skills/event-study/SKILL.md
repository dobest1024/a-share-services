# 事件研究法

## 核心概念

测算事件发生前后窗口期内股价的异常收益（AR）与累计异常收益（CAR）。

## 标准流程

### Step 1 事件定义
- 事件类型（业绩超预期 / 股东增持 / 监管函 / 重组 / 解禁 ...）
- 事件日 = 公告日 t=0（A 股以公告日为准，停牌时取复牌日）

### Step 2 样本筛选
- 时间范围：近 5-10 年
- 排除：ST、新股、停牌过长、缺失数据

### Step 3 估计期 vs 事件窗口期
- 估计期：[t-180, t-30] 用于估计市场模型参数
- 事件窗口：[t-20, t+60] 或 [t-5, t+20]（看事件类型）

### Step 4 异常收益计算
**市场模型（最常用）**：
```
R_it = α_i + β_i × R_mt + ε_it    （估计期回归）
AR_it = R_it - (α̂_i + β̂_i × R_mt)  （事件窗口异常收益）
CAR_i[τ1, τ2] = Σ AR_it
```

基准：沪深 300 / 中证 500（按个股板块选择）。

### Step 5 横截面统计
- 平均 CAR
- t 统计量（CAR 是否显著 ≠ 0）
- 中位数 CAR（剔除极端值）
- 正例比例（% of CAR > 0）

### Step 6 分组对比（可选）
按公司特征分组看异常收益差异：
- 市值（大盘 vs 小盘）
- 行业
- 估值（高 PE vs 低 PE）
- 之前 6 月动量

## A 股事件研究常见课题

| 事件 | 典型窗口 | 历史规律（参考）|
|---|---|---|
| 业绩超预期 50% | [-5, +20] | 平均 CAR ~+8%，但分化大 |
| 大股东增持 > 1% | [-3, +30] | 平均 CAR ~+5% |
| 高管减持 > 1% | [-3, +30] | 平均 CAR ~-3% |
| 监管函 | [0, +20] | 平均 CAR ~-5% |
| 重组停牌 | [0, +60] | 高度分化，依重组类型 |
| 解禁日 | [-20, +30] | 平均 CAR ~-2% |
| 业绩预告"扭亏" | [0, +30] | 平均 CAR ~+10% |

⚠️ 历史规律仅供参考。具体事件需独立分析，市场环境不同表现差异巨大。

## 工作流

1. 从公告库 / `cninfo-mcp` 拿事件清单
2. 配对行情数据（`akshare-mcp.stock.daily` 复权后）
3. 估计市场模型参数
4. 算 AR / CAR
5. 横截面汇总 + 分组
6. 输出报告

## 输出结构

```
## 事件研究 - {事件类型}

### 样本
- 事件数：N
- 时间窗口：[t-X, t+Y]
- 基准：沪深 300
- 模型：市场模型

### 平均异常收益
| 日 | 平均 AR | 累计 CAR | t 值 |
| -5 | ... |
| -1 | ... |
| 0 | ... |
| +1 | ... |
| ... |
| +20 | ... |

### 横截面统计
- 平均 CAR[0, +20]：+X.X%（t = X.X）
- 中位 CAR：+X.X%
- 正例占比：X%

### 分组结果
按市值 / 行业 / 估值 / 历史动量分组的 CAR 差异

### 应用建议
- 该事件作为信号的有效性
- 失效情境
- 与其他事件 / 因子组合的可能性

### 注意
- 事件研究测的是**平均效应**，单个事件不能保证表现
- 不构成投资建议
```
