# 行业分类

## 默认口径：申万

A 股研究、卖方策略、行业基金分类，**统一对外用申万**。
- 一级：31 个（如：电力设备、电子、医药生物）
- 二级：134 个
- 三级：346 个

行情、估值、资金流的"行业"维度都按申万一级聚合，便于横向对比。

## 切换口径的情形

| 场景 | 用哪个 |
|---|---|
| 默认 | 申万 |
| 与海外资产对比 | GICS（11 大类）|
| 政策研究（央国企、专项行动）| 证监会大类（19 类）|
| 中信系研究所对接 | 中信（30 个一级，与申万对齐度高）|

## 映射表

参考 `reference-data/industry-map.csv`，按季度更新。映射工作原则：

1. 主营业务收入占比 > 50% 的业务对应行业即为公司归属
2. 跨界经营公司，按主营 + 标签处理（如比亚迪：申万一级"汽车" + 标签"动力电池"）
3. 控股型公司按"综合"归类，但展示子公司业务结构

## 查询方式

- 单只：`akshare-mcp.stock.profile` 返回 `sw_l1 / sw_l2 / sw_l3`
- 行业成分股：`akshare-mcp.stock.sw_index_constituents`
- 行业历史估值：`akshare-mcp.stock.sw_valuation`

## 常见陷阱

- **创业板** / **科创板** 不是行业，是板块；不要用作行业聚合维度
- 申万的 2021 版与 2014 版差异大（如新增"电力设备"独立一级），跨期对比要注意口径切换
- 北交所部分公司在 akshare 中行业字段为空，需用证监会代码兜底
