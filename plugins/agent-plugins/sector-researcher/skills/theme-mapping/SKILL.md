# 主题映射

## 映射逻辑

主题映射不是简单的"概念股清单"，而是结构化的：
```
主题 → 受益行业 → 子行业 → 产业链位置 → 代表公司
```

## 主要数据源

- `reference-data/policy-keywords.yaml`（主题 → 行业映射）
- akshare 概念板块（实时市场跟踪的概念）
- Wind / Choice 概念板块（机构口径，更稳定）

## A 股主题分类

### 政策驱动主题
- 新质生产力
- 中特估（中国特色估值体系）
- 专精特新（小巨人）
- 双碳（碳达峰碳中和）
- 国产替代
- 一带一路
- 数字经济
- 乡村振兴
- 老龄化 / 银发经济

### 产业驱动主题
- 人工智能（含算力 / 大模型 / 应用）
- 新能源汽车（含整车 / 三电 / 智能驾驶）
- 半导体（含设计 / 制造 / 设备 / 材料）
- 创新药 / CXO
- 高端制造（机床 / 工业机器人）
- 储能（电化学 / 抽蓄 / 氢能）
- 卫星互联网
- 量子科技

### 周期 / 复苏主题
- 消费复苏
- 地产链
- 服贸出口
- 大宗商品

### 短期事件主题
- 重磅会议（两会 / 政治局 / 经济工作会议）窗口
- 业绩高增长期
- 国际地缘（如俄乌、巴以、台海）

## 主题判断框架

| 维度 | 评估 |
|---|---|
| 政策支撑 | 强 / 中 / 弱 |
| 产业基础 | 已有产业 / 萌芽 / 概念 |
| 业绩兑现 | 已兑现 / 预期 / 概念 |
| 持续周期 | 长期（>2 年）/ 中期 / 短期（炒一波）|
| 资金共识 | 机构共识 / 游资 / 散户 |

## 输出结构

```
## 主题映射 - {主题名}

### 主题定义
- {一句话）
- 政策依据（关键文件）
- 产业基础（已存在还是概念）

### 主题维度评估
- 政策支撑：★★★★★
- 产业基础：★★★
- 业绩兑现：★★
- 持续周期：长期
- 资金共识：机构 + 游资

### 受益行业（按确定性排序）
| 申万一级 | 受益逻辑 | 子行业重点 |

### 产业链结构
```
上游：{原料 / 设备}
中游：{制造 / 关键技术}
下游：{应用}
```

每个环节列代表公司（按市值 / 行业地位）：
| 环节 | 公司 | 代码 | 主营 | 市值 |

### 标的池
- 龙头股（机构共识）：N 只
- 二线 / 细分龙头：N 只
- 弹性标的（小盘 + 高题材）：N 只
- 高股息 / 价值兼具：N 只（如适用）

### 风险
- 政策不及预期
- 业绩不能兑现
- 已被高估
- 主题轮动到其他方向

### 跟踪指标
- 行业数据
- 政策落地节奏
- 龙头股股价 / 估值
- 北向 / 主力资金动向
```
