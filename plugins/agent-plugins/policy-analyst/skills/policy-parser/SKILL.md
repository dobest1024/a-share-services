# 政策文件解读

## 主要政策来源

| 部委 | 关注领域 |
|---|---|
| 国务院 | 顶层文件、五年规划、政府工作报告 |
| 中央政治局 | 重要会议公报、定调宏观方向 |
| 中央经济工作会议 | 年度经济政策定调 |
| 央行（人民银行）| 货币政策、利率、汇率、金融监管 |
| 证监会 | 资本市场监管、IPO、再融资、退市、并购 |
| 国家发改委 | 产业政策、价格、新型基础设施 |
| 工信部 | 制造业、信息化、新材料、新能源 |
| 国家能源局 | 电力、油气、新能源 |
| 国家药监局 | 医药、医疗器械 |
| 财政部 | 财税、减税降费、政府支出 |
| 国资委 | 央国企改革 |
| 银保监会（金融监管总局）| 银行、保险监管 |

## 文件级别

| 级别 | 例 |
|---|---|
| 法律 / 党中央国务院文件 | 中共中央 国务院关于 ... |
| 部委规章 / 通知 | 国务院办公厅关于 ... / XX 部令 |
| 行业规划 | "十四五" XX 规划 |
| 部门通知 / 指导意见 | 关于 ... 的通知 / 指导意见 |
| 答记者问 / 新闻发布会 | 较低法律效力但传递信号 |

## 解读维度

### 1. 文件性质
- 强制 vs 引导
- 全局 vs 局部
- 长期 vs 短期

### 2. 主要内容
- 关键术语（用 `reference-data/policy-keywords.yaml` 映射）
- 政策目标
- 工具与手段
- 时间表

### 3. 受益方向
- 行业（通过主题映射表）
- 公司类型（央国企 / 民企 / 外资 / 中小企业）
- 区域

### 4. 历史可比文件
- 类似政策的历史落地节奏
- 类似政策的市场反应模式

### 5. 落地节奏
- 短期（< 6 月）：交易型机会
- 中期（6-24 月）：业绩兑现
- 长期（> 2 年）：产业格局变化

## 关键陷阱

### 已经定价
- 多数政策在正式发布前已有预期
- 看市场对发布日的反应（"预期内 / 预期外 / 不及预期"）

### 落地不及预期
- 政策表述与实际细则差距大
- 财政补贴等需配套细则才能受益

### 政策反复
- 部分政策曾有反复（如 K12 教育 / 互联网平台）
- 不要假设政策线性递推

## 工作流

1. 输入：政策文件链接 / 文本
2. 提取：发文方、级别、关键术语、主要内容
3. 映射受益主题与行业（`reference-data/policy-keywords.yaml`）
4. 历史可比文件检索
5. 落地节奏判断
6. 输出"政策摘要 + 受益方向 + 节奏 + 风险"
