# 持仓估值复核

## 复核重点

### 1. 停牌股估值
- 短期停牌（< 5 个交易日）：按停牌前最后收盘价
- 长期停牌（重大重组 / 立案）：
  - 估值技术：行业指数法（用申万一级指数复权调整）
  - 估值技术：可比公司法
  - 估值技术：第三方估值机构
- 关键：估值方法须在基金合同中提前约定

### 2. 涨跌停股估值
- 当日涨停 / 跌停封单 → 按收盘价估值
- 关键：警惕"看似封死但次日开板"的流动性陷阱
- 持仓占自由流通比例高时需做流动性折扣评估

### 3. 限制流通股
- 限售股按对应流通股折扣（如折扣 15-25%）
- 锁定到期前后估值调整

### 4. 非标资产
- 私募基金持有非上市股权 / 债权
- 估值方法：成本法 / 第三方评估 / 模型估值
- 至少每年一次重新评估

### 5. 衍生品
- 股指期货：盯市（每日结算价）
- 股指期权：BS / Binomial 模型
- 场外期权：第三方报价 / 自建模型

### 6. 港股通持仓
- 按当日港股收盘价
- 汇率取中间价

## 公允性测试

| 测试 | 方法 |
|---|---|
| 价格合理性 | 与最近成交价对比、与同行业可比标的对比 |
| 流动性折扣 | 持仓占自由流通比例 → 应有折扣 |
| 估值一致性 | 同一标的不同产品估值应一致 |
| 历史回顾 | 估值与后续实现价格的偏差 |

## A 股特殊场景

### 长期停牌（半年以上）
- 必须使用替代估值方法
- 不能继续用停牌前价
- 估值偏差大时记入估值储备

### 立案调查 / ST 警示
- 立案当日 / 次日通常跌停
- 估值需考虑后续连续跌停的可能
- 流动性大幅恶化 → 流动性折扣

### 财务造假退市
- 退市整理期：按 ST 涨跌幅
- 退市进入老三板：估值大幅折让

## 工作流

1. 输入：基金持仓
2. 标记需要特别复核的标的
3. 逐个核对估值方法的适用性
4. 估算公允价值范围
5. 输出"复核报告 + 异常清单"

- 净值层面偏差估计：±X%
