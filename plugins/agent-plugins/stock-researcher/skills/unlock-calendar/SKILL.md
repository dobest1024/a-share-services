# 限售解禁分析

## A 股限售股类型

| 类型 | 锁定期 | 解禁后 |
|---|---|---|
| 首发原始股 | 1 年（控股股东 3 年）| 全流通 |
| 定增（小于 35 元小于）| 6 个月（询价）/ 18 个月（锁定）| 全流通 |
| 股权激励限制性股票 | 12-36 个月分批 | 解锁后流通 |
| 股权激励期权 | 等待期满即可行权 | 行权日流通 |
| 大股东可交换债换股 | 换股日 | 立即流通 |

## 解禁压力测算

### 绝对量
- 解禁数量 / 解禁市值
- 解禁市值 / 自由流通市值 → 真实冲击

### 解禁性质
| 性质 | 卖出概率 | 备注 |
|---|---|---|
| 首发限售（创投/机构股东）| 高 | 创投基金有退出诉求 |
| 控股股东首发限售 | 低 | 5% 以上股东减持受新规约束 |
| 定增机构投资者 | 高-中 | 跟着浮盈情况，浮盈大时减持概率高 |
| 员工持股 / 股权激励 | 中 | 看个人意愿 |
| 战略投资人 | 低 | 通常是产业链合作 |

### 浮盈状态
- 当前股价 vs 定增价 → 定增投资者浮盈
- 浮盈 > 50%：减持动机强
- 浮亏：可能等待回本

## 解禁前后股价表现（A 股历史模式）

- T-30 ~ T-15：**预期压制**，股价相对走弱
- T-15 ~ T-5：**抛售试探**，部分股东大宗或集合竞价提前出货
- T-5 ~ T+5：**情绪低点**，往往是阶段底
- T+5 ~ T+30：减持节奏决定，部分公司公告"承诺不减持"形成反弹

⚠️ 上述是统计规律，非个股必然走势。

## 数据接口

- `akshare-mcp.event.unlock_calendar(start_date, end_date)` — 全市场解禁日历
- `akshare-mcp.event.unlock_stock(code)` — 个股解禁历史与未来安排

## 输出结构

```
## 限售解禁 - {标的}

### 未来 6 个月解禁
| 解禁日 | 解禁股数 | 解禁市值 | 占自由流通比 | 类型 | 股东 |

### 关键解禁事件深度分析
对每笔 > 10% 自由流通市值的解禁，展开：
- 解禁股东背景（创投 / 战投 / 员工持股）
- 取得成本（定增价 / 首发价）
- 当前浮盈
- 历史减持记录

### 历史解禁后表现
- 该标的过去解禁日 ±30 日股价表现统计

### 关注要点（不是建议）
- 大解禁前是否有"承诺不减持"公告
- 解禁后是否有立即出现的大宗交易
- 板块整体解禁压力（同业是否集中解禁）
```
