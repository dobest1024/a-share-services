# 催化剂日历

## 催化剂分类

### 公司层面
- 财报披露日（季报 / 中报 / 年报 / 业绩快报）
- 业绩预告窗口
- 限售解禁日
- 重大资产重组关键节点（停牌 / 预案 / 草案 / 股东大会 / 证监会受理 / 实施）
- 股东大会
- 业绩说明会 / 投资者交流日
- 分红除权日
- 重要产品发布 / 新工厂投产
- 主要合同签署 / 客户验证

### 行业层面
- 月度 / 季度行业数据发布（销量 / 出货量 / 库存）
- 行业协会会议
- 国际重要展会 / 大会（如 CES / 慕尼黑车展 / 中国国际医药健康大会）
- 主要海外同行财报

### 宏观层面
- 央行议息会议 / 公开市场操作窗口
- 经济数据发布（CPI / PMI / 工业增加值 / 社融 / 出口）
- 重要会议（两会 / 中央经济工作会议 / 政治局会议 / 三中全会）
- 重要部委文件预期窗口

## 重要性等级

| 等级 | 标准 |
|---|---|
| ★★★ | 可单独决定股价方向（业绩雷 / 重大重组 / 关键政策） |
| ★★ | 中短期重要影响 | 
| ★ | 一般关注 |

## 工作流

1. 输入：股票池 / 关注行业 / 时间窗口（默认未来 30 天 + 关键 90 天）
2. 拉取：
   - 财报披露安排（`akshare-mcp.event.earnings_calendar`）
   - 解禁日历（`akshare-mcp.event.unlock_calendar`）
   - 分红日历（`akshare-mcp.event.dividend_calendar`）
   - 重组进度（`cninfo-mcp.filings_list` + 重组类型筛选）
   - 经济数据日历（`akshare-mcp.macro.calendar`）
3. 标注重要性等级
4. 按日期排序
5. 标注潜在影响方向（中性 / 偏多 / 偏空）— 限于公开信息推断

## 输出

```
## 催化剂日历 - {开始日}至{结束日}

### 本周（高频跟踪）
| 日期 | 标的 / 主题 | 事件 | 等级 | 备注 |

### 下周
（同结构）

### 未来 2-4 周（中频）
（同结构）

### 未来 1-3 月（提前关注）
（同结构）

### 风险窗口聚焦
- 财报披露季压力测算
- 解禁集中区间
- 重要政策窗口（如两会、年中政治局）

### 跟踪建议
- 按事件等级安排跟踪频率
- 高等级事件提前 1-2 周深度准备
```
