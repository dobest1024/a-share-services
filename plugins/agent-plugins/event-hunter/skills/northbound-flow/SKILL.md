# 北向资金分析

## 北向资金的两个通道

- **沪股通**：港股投资者买卖上交所标的（不含科创板部分标的）
- **深股通**：港股投资者买卖深交所标的
- 合称"北向"，又称"陆股通"，俗称"外资"

每日额度：每个通道 520 亿元净买入上限（实际从未触及，但作为信号关注）。

## 数据频率与披露

| 频率 | 内容 | 时点 |
|---|---|---|
| 实时 | 北向当日净买入额 | 交易时段 |
| 日终 | 北向当日 Top 10 活跃股 | 当日 18:00 |
| 日终 | 北向所有持股数（按个股）| 次日 9:00 前 |

⚠️ 2024 年 8 月起 **取消北向资金实时披露**，仅披露日终持股数。实时净买入数据不可用。

## 分析维度

### 1. 总量
- 当日净买入 / 卖出（注意：现在是 T+1 披露）
- 5 日 / 20 日累计
- 当年累计

### 2. 个股层面
- 北向持股市值 / 自由流通市值 → "北向占流通比"
  - > 10% 为重仓
  - > 15% 接近持股上限警戒（外资单一股票上限 30%）
- 近 5 日 / 20 日北向持股变动（绝对值 + 占比变化）
- 北向加仓 / 减仓速度（环比变化）

### 3. 行业层面
- 北向最新行业偏好（占自由流通市值比）
- 偏好变化：与上月对比加减仓最多的行业
- 历史：北向偏好食品饮料、家电、医药、新能源；近年减配地产、银行（但 2023 后中特估行情有所修复）

### 4. 风格判断
- **北向重仓股 ≠ 北向"主动买"**：大量是被动配置（指数基金）
- 单日大额净买入≠看多：可能是被动指数调仓、可能是 H 股 / A 股套利、可能是衍生品对冲
- 北向连续 5-10 日净买入某行业 → 风格信号较可靠

## 数据接口

- `akshare-mcp.fund_flow.northbound_daily()` — 北向日终净买入与累计
- `akshare-mcp.fund_flow.northbound_holdings(code)` — 个股北向持股历史
- `akshare-mcp.fund_flow.northbound_top10()` — 当日 Top 10 活跃股
- `akshare-mcp.fund_flow.northbound_industry()` — 北向行业偏好

## 输出结构

```
## 北向资金 - {标的}

### 持仓快照
- 北向持股数 = X 万股
- 持股市值 = X 亿
- 占自由流通市值 = X.X%
- 历史分位（近 2 年）= X 分位

### 变动趋势
- 近 5 日变动 = +X 万股（+X%）
- 近 20 日变动 = +X 万股（+X%）
- 近 1 年高低点 = X / X 万股

### 行业上下文
- {标的所在申万一级} 北向偏好 = X%（行业内自由流通市值的占比）
- 近 1 月行业偏好变化 = +X% / -X%

### 风格判断
- 持续加仓 / 见顶减仓 / 区间震荡 / 配置型稳定

### ⚠️ 解读限制
- 2024 年 8 月起北向实时数据不再披露，本分析为 T+1 数据
- 北向 ≠ 主动外资判断，部分为被动指数 / 套利 / 对冲
```
