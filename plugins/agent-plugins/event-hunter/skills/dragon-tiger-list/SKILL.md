# 龙虎榜分析

## 触发条件（交易所规则）

| 触发 | 主板 | 创业板 / 科创板 |
|---|---|---|
| 日涨跌幅偏离值（异常）| ±7% | ±15% |
| 日振幅 | 15% | 30% |
| 日换手率 | 20% | 30% |
| 连续 3 日累计涨跌幅 | ±20% | ±30% |

满足任一即上榜，披露当日前 5 名买入营业部 + 前 5 名卖出营业部。

## 席位类型识别

### 机构席位
- 名称含"机构专用"
- 含义：公募 / 私募 / 保险 / 社保等买方机构（无法识别具体机构，但能区分于游资）
- 一笔机构净买入通常代表配置性买入

### 知名游资席位（**仅做风格识别，不构成跟随建议**）
- **拉萨团结路** / **拉萨东环路** / **拉萨金融城南环路**：东方财富券商西藏分部，散户/小游资聚集
- **章盟主**：方新侠系，喜欢大额抢筹热点龙头
- **作手新一**、**孙哥**、**赵老哥** 等具名游资：通过历史龙虎榜数据识别其常用席位
- **量化席位**：高频换手，量化席位的买卖一致性高、出现在多只个股

### 北向席位
- "香港中央结算（深股通/沪股通）" → 北向资金

## 分析维度

### 1. 买卖力度
- 净买入 = 买入总额 - 卖出总额
- 占当日成交比例 → 龙虎榜资金"主导力度"

### 2. 席位类型组合
| 组合 | 解读 |
|---|---|
| 机构 + 机构买入 | 配置型机构入场，相对持续 |
| 游资 + 游资买入 | 短线博弈，警惕次日波动 |
| 机构卖 + 游资买 | "机构出货游资接力"，短线但下一周可能高位 |
| 北向买入 + 机构买入 | 核心资产共识 |
| 多家游资分散卖出 | 形态破坏，警惕 |

### 3. 历史关联
- 同一席位最近 30 日还买了哪些股 → 风格判断
- 同一席位历史买入后 1 / 3 / 5 / 10 日平均涨跌幅

### 4. 龙虎榜后表现复盘
- 上榜次日开盘是否高开 / 低开
- 5 日 / 10 日累计涨跌幅
- 是否再次上榜（连续上榜代表资金分歧大）

## 数据接口

- `akshare-mcp.fund_flow.dragon_tiger(date)` — 当日龙虎榜全市场
- `akshare-mcp.fund_flow.dragon_tiger_stock(code, lookback_days)` — 个股最近龙虎榜历史
- `akshare-mcp.fund_flow.lhb_seat_history(seat_name, days)` — 席位历史买入

## 输出结构

```
## {日期} 龙虎榜 - {股票}

### 上榜原因
- 触发：日涨幅 X%（板块阈值 X%）
- 当日成交 X 亿，换手率 X%

### 买入前 5 席位
| 排名 | 席位 | 买入额 | 占总成交 | 席位类型 | 历史风格 |

### 卖出前 5 席位
（同结构）

### 资金主导力度
- 龙虎榜净买入 = X 亿（占当日成交 X%）
- 机构席位净买入 = X 亿

### 风格判断
- 主导方：游资 / 机构 / 北向 / 量化（综合判断）
- 风格标签：龙头加速 / 趋势接力 / 高位分歧 / 调整低吸 ...

### 关注要点（不是建议）
- 次日开盘观察点
- 关联个股（同席位近期布局）

### ⚠️ 风险提示
- 龙虎榜后表现历史均值仅供参考，不代表本次走势
```
