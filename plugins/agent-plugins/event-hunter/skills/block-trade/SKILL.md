# 大宗交易分析

## 基础概念

大宗交易：交易所盘后撮合的大额成交，规则：
- 单笔成交：A 股主板 ≥ 30 万股 或 200 万元
- 价格区间：当日收盘价 ±10%（创业板 / 科创板 ±20%）
- 不计入分时成交，但影响次日参考价

## 解读维度

### 折溢价率
```
折溢价率 = (大宗成交价 - 当日收盘价) / 当日收盘价
```

| 折溢价 | 常见含义 |
|---|---|
| < -5% | 大幅折价，**减持迫切**（限售股大股东、机构清仓） |
| -3% ~ -5% | 中度折价，机构出货 / 大股东减持 |
| -1% ~ -3% | 小幅折价，常规过户 |
| ±1% | 平价，关联方过户 / 调仓 |
| +1% ~ +5% | 溢价，**承接方愿意付溢价**（看多信号）|
| > +5% | 大幅溢价，定向收购 / 战略入股 |

### 买卖方营业部
- 卖方为大股东 / 限售解禁后的减持渠道（最常见）
- 买方为机构席位 → 大资金接力
- 买方为知名游资 → 短线博弈
- 双方为同一系营业部 → 关联过户、需警惕利益输送

### 数量与连续性
- 单日大宗 vs 连续大宗（连续 3-5 日有大宗 → 减持节奏）
- 占当日成交比例（> 10% 信号意义大）

## A 股特有场景

### 减持新规约束
- 大股东、董监高减持需提前公告
- 大宗交易接盘方需锁定 6 个月（防止"过桥减持"）
- 实控人 / 控股股东减持有更严约束

### 限售股大宗渠道
- 限售解禁后大股东最常用大宗减持（避免二级市场冲击）
- 解禁日前后大宗交易突增是常见模式

### 战略增持
- 持续溢价大宗 + 买方为产业资本 / 战投 → 潜在收购
- 关注后续公告（要约 / 协议转让 / 一致行动）

## 数据接口

- `akshare-mcp.fund_flow.block_trade_daily(date)` — 当日全市场大宗
- `akshare-mcp.fund_flow.block_trade_stock(code, days)` — 个股大宗历史

## 输出结构

```
## 大宗交易 - {标的} 近 30 日

### 概况
- 总成交笔数 = N
- 总成交金额 = X 亿
- 平均折溢价率 = X%

### 明细（最近 10 笔）
| 日期 | 成交价 | 收盘价 | 折溢价 | 数量 | 买方席位 | 卖方席位 |

### 模式识别
- 主导特征：连续折价减持 / 间歇平价过户 / 战略溢价增持
- 与限售解禁的时间关系
- 买卖方席位类型分布

### 关注要点
- 是否需要交叉核对减持公告
- 是否有关联交易嫌疑
- 后续若有连续大宗，需提高跟踪频率
```
