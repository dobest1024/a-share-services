# 公司行动事件

## 五类常见公司行动

### 1. 定向增发（再融资）
- **询价 vs 锁定**：询价 6 月锁定，定价 18 月锁定
- 关注：发行对象（战投 / 财投 / 大股东参与）、发行价（折扣比例）、募资用途
- 信号：
  - 大股东参与 → 看多
  - 知名外部投资人参与 → 中性偏多
  - 募资还债 / 补流 → 偏空（缺钱）
  - 募资扩产 / 收购 → 看业务方向

### 2. 回购
- **回购用途**：注销（最积极，缩股本）/ 员工持股 / 股权激励
- **回购方式**：集中竞价 / 要约 / 大宗
- **金额测算**：回购金额 / 总市值 → 力度
- 信号：
  - 注销式回购 + 大额 → 强看多
  - 员工持股回购 + 大额 → 看多
  - 小额象征性回购 → 中性
  - 回购公告但执行率低 → 警惕（喊口号）

### 3. 减持
- **减持新规**：5% 以上股东 90 天内减持不超 1%（集中竞价）/ 2%（大宗）
- 关注：减持人身份（控股股东 / 实控人 / 一致行动人 / 财务投资人）
- 信号：
  - 控股股东减持 → 利空（信号最强）
  - 高管减持 → 利空中等
  - 创投股东减持 → 中性（退出诉求）
  - "因个人资金需求" → 通用理由，要结合上下文

### 4. 重大资产重组
- 类型：吸收合并 / 资产置换 / 发行股份购买资产 / 现金收购
- 阶段：停牌 → 预案 → 草案 → 股东大会 → 证监会审核 → 实施
- 关注：
  - 标的资产质量与估值
  - 业绩承诺 vs 标的盈利能力
  - 商誉形成压力
  - 与控股股东的关联性（同业竞争解决？资产注入？）

### 5. 股东大会决议
- 普通决议（多数赞成）：年度报告 / 利润分配 / 董事会换届
- 特别决议（2/3 赞成）：修改章程 / 增减资 / 重大资产重组 / 合并分立
- 关注：
  - 中小股东投票分歧（反对率 > 10% 警惕）
  - 关联方回避情况
  - 累积投票制下董事提名

## 工作流

1. 拿到公告（`cninfo-mcp.filing_text(url)`）
2. 识别类型（5 类之一）
3. 抽取关键参数（金额 / 价格 / 比例 / 时点）
4. 与该公司历史同类事件对比
5. 输出"事件分类 + 影响判断 + 后续观察"

## 输出结构（以定增为例）

```
## 定向增发公告 - {标的}

### 基本要素
- 发行方式：询价 / 定价
- 发行对象：{特定对象，数量 N 个}
- 发行价：X 元/股（较公告前 20 日均价折扣 X%）
- 发行数量：X 万股（占发行后总股本 X%）
- 募资总额：X 亿
- 锁定期：6 月 / 18 月

### 募资用途
- 项目 1：X 亿（XX 项目，IRR 预期 X%）
- 项目 2：X 亿
- 补充流动资金：X 亿（占比 X%，监管要求 < 30%）

### 信号判断
- 是否有大股东参与？
- 外部投资人是否知名？（产业资本 / 公募 / 险资 / 知名私募）
- 募资是否聚焦扩张？

### 历史对照
- 公司过去 3 次再融资执行情况
- 同行业近期定增定价水平

### 关注要点
- 证监会批文进度
- 询价定价区间
- 定增完成后大股东持股比例稀释
```
