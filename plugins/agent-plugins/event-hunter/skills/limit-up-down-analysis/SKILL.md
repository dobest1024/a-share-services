# 涨跌停分析

## 板块涨跌停规则（再次提醒）

| 板块 | 涨跌停 |
|---|---|
| 主板 | ±10% |
| 创业板 / 科创板 | ±20% |
| 北交所 | ±30% |
| ST | ±5% |
| 主板新股首日 | ±44% |

## 涨停核心概念

### 封板强度
- **封板金额**：涨停价买入挂单未成交的总金额
- **封单比**：封板金额 / 流通市值
  - > 1% 强封
  - 0.3-1% 中
  - < 0.3% 弱封，易炸板
- **首次涨停时间**：早盘涨停 > 午后涨停 > 尾盘涨停（一般来说）

### 连板梯队
- 1 板：当日新晋（最多）
- 2 板：次日继续涨停（晋级率统计）
- 3 板 / 4 板：高度板，资金分歧加大
- 5 板+：龙头股，但顶部风险骤增

### 炸板
- 炸板：盘中涨停被打开（开板）
- 炸板后封回：强势（资金回流）
- 炸板后未封回：弱势（情绪降温）

## 复盘分析维度

### 1. 涨停原因
- 业绩超预期
- 行业政策催化
- 主题概念（消息驱动）
- 龙虎榜接力（资金博弈）
- 重组 / 资产注入预期
- 技术形态突破

### 2. 板块联动
- 同概念涨停家数
- 涨停潮（>30 家）vs 零星涨停（<10 家）
- 龙头股 vs 跟风股识别

### 3. 次日表现统计
| 状态 | 次日开盘 |
|---|---|
| 一字涨停（开盘即封）| 高开 5-8% 概率高 |
| 强封 + 早盘涨停 | 平开或小高开 |
| 尾盘弱封 | 平开或小低开 |
| 炸板 | 低开概率高 |

## 数据接口

- `akshare-mcp.fund_flow.limit_up_pool(date)` — 当日涨停股池
- `akshare-mcp.fund_flow.limit_up_history(code)` — 个股涨停历史
- `akshare-mcp.fund_flow.continuous_up_ranking(date)` — 当日连板梯队

## 工作流

1. 拿到当日涨停股池
2. 按概念聚合，识别主线
3. 对龙头股展开分析（封板强度 + 资金 + 题材）
4. 输出"主线 + 龙头 + 接力候选 + 风险信号"

## 输出结构

```
## {日期} A 股涨停复盘

### 全市场
- 涨停家数：N（剔除一字 / 新股）
- 跌停家数：M
- 炸板率：X%
- 连板梯队：N1 板 / N2 板 / N3+ 板

### 主线识别
| 主线 | 涨停家数 | 龙头 | 触发事件 |

### 龙头股深度
对每条主线龙头：
- 板数 / 封板时间 / 封单金额
- 资金动向（龙虎榜 / 北向 / 两融）
- 触发逻辑
- 风险点

### 次日观察清单
- 关注主线持续性
- 关注龙头分歧（高度板能否承接）
- 关注炸板回封 / 跌停反包反转

### ⚠️ 提示
- 短线博弈风险极高，本分析仅做市场情绪复盘
- 不构成任何买卖建议
```
