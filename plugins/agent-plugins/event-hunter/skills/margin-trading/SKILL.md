# 融资融券分析

## 基础概念

- **融资余额**：投资者借券商钱买股票，未还的本金。**看多**杠杆。
- **融券余额**：投资者借券商股票卖出，未还的股票市值。**看空**杠杆。A 股融券规模小、限制多。
- **两融余额**：融资 + 融券，但**主要看融资余额**（融券占比通常 < 5%）。

## 关键指标

### 全市场
- 两融余额总量 = 重要市场情绪指标
  - 1.5 万亿以下：风险偏好低
  - 1.5-1.8 万亿：常态
  - 1.8 万亿以上：风险偏好高
- 增速：日变化 + 周变化 + 月变化

### 个股
- 融资余额 / 流通市值 → "融资盘占比"
  - > 5%：杠杆资金活跃
  - > 8%：杠杆较重，警惕双向波动放大
  - > 10%：风险警戒
- 融资买入额 / 当日成交 → "融资买入占比"
  - > 15%：融资盘是当日主导力量

## 强平警戒

- 融资保证金维持比例 = (现金 + 信用账户证券市值) / 融资负债
- 警戒线 130%，强平线 110%（不同券商有微调）
- 标的下跌 → 维持比例下降 → 触发追加保证金或强平
- 高融资占比的股票，下跌时易引发**连锁强平**（俗称"杠杆踩踏"）

## A 股两融特点

- 可融资标的范围有限：沪深 300 + 中证 500 + 中证 1000 + 部分行业指数成分（约 2000 只）
- 创业板 / 科创板的两融比沪深主板灵活度低
- 北交所 2023 起开放融资融券
- 转融通制度让公募 / 险资可以出借券给融券客户

## 数据接口

- `akshare-mcp.fund_flow.margin_total()` — 全市场两融余额时间序列
- `akshare-mcp.fund_flow.margin_stock(code)` — 个股两融余额历史
- `akshare-mcp.fund_flow.margin_daily(date)` — 当日全市场融资融券变化

## 输出结构

```
## 融资融券 - {标的}

### 当前状态
- 融资余额 = X 亿
- 融资余额 / 流通市值 = X.X%（历史分位 X%）
- 融券余额 = X 亿（占比 X%）

### 近期变化
- 近 5 日融资净买入 = +/-X 亿
- 近 20 日趋势 = 持续加仓 / 净减仓 / 震荡

### 与行业对比
- 申万二级行业内融资占比中位 = X%
- 标的在行业内排序 = N / 总数

### 杠杆风险信号
- 融资占比是否高于 8%（杠杆警戒）
- 近期是否有快速增长（短期投机信号）
- 标的近 20 日波动率 → 强平触发概率
```
